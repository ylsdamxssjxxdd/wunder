# wunder

一句话介绍：wunder 是一个能在你授权后**完全接管电脑、替你办公**的桌面智能体。现在主推 **wunder-desktop** —— 下载、安装、打开就能用。

它不只是聊天，而是“会做事”的智能体：你只要说目标，它会自动拆解任务、调用工具、产出结果；需要时还能 **多智能体并行协同**，多个智能体分工同时完成不同环节。

## 核心能力一览（结合 wunder 现有能力）
| 能力 | 具体说明 | 常见用途 |
| --- | --- | --- |
| 桌面控制与自动化 | 本地桌面形态可在授权后操作应用、文件与网页 | 整理资料、批量处理、生成文档 |
| MCP 工具体系 | 支持 /wunder/mcp，自托管与跨系统工具接入 | 连接外部服务、跨系统调用 |
| Skills 流程化能力 | 以技能的形式固化流程与步骤 | 报告生成、固定流程一键执行 |
| 多智能体并行协作 | 多个智能体并行、互相交接 | 一个查资料、一个写稿、一个复核 |
| 定时任务与调度 | 网关/渠道/定时任务能力支持长期自动化 | 定时整理、自动提醒、周期任务 |
| 长会话与记忆 | 上下文压缩 + 长期记忆 | 长期项目、持续迭代 |
| 统一接入与渠道 | server 统一入口，WS 优先 + SSE 兜底 | 多端一致体验、稳定接入 |

## 你可以用它做什么
- 写文档、做方案、整理材料、生成表格与演示
- 资料搜索与总结、要点提炼、会议纪要
- 写代码、重构、生成脚本与自动化流程
- 组合 MCP + Skills 打造你的专属流程
- **构建 wunder 本身，实现自举**（用 wunder 写代码、补文档、跑任务）

## 快速开始（3 步）
1. 打开项目 Releases 页面，下载与你系统匹配的 wunder-desktop 安装包（以发布页为准）。
2. 安装/解压并启动 wunder-desktop。
3. 在输入框里直接描述你的任务，回车即可。

小提示：你只想体验的话，到这一步就够了。

## 三种形态，按需选择
| 形态 | 适合谁 | 一句话 |
| --- | --- | --- |
| **wunder-desktop（推荐）** | 普通用户 / 个人 | 下载即用，本地桌面智能体 |
| `wunder-server` | 团队 / 组织 | 统一接入、权限管理、多租户与协作 |
| `wunder-cli` | 开发者 / 自动化 | 命令行驱动、脚本化任务执行 |

> 只想快速上手？用 **wunder-desktop**。需要团队协作与系统接入？再了解 server。需要自动化脚本？选 cli。

## 常见问题
**Q: 我需要先部署 server 吗？**  
A: 不需要。wunder-desktop 是主推版本，下载后直接使用即可。只有当你需要团队协作或更复杂的接入时，才需要考虑 server。

**Q: 我不懂技术也能用吗？**  
A: 可以。它是面向普通用户设计的，只要“说目标”，智能体会帮你完成。

**Q: 多智能体怎么用？**  
A: 你只需要提出目标，系统会自动分工并行处理；你也可以在对话中要求它“并行完成”。
