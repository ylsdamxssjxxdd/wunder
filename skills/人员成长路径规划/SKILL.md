# 人员成长路径规划

## 概要
收集人员现状与目标信息，结合组织内外部能力模型与数据源，对标差距并给出分阶段成长路径与行动建议，输出可执行的规划报告与图表。

## 快速流程
1. 收集人员信息、经历、能力评估与目标岗位/方向。
2. 检索知识库/数据库（岗位能力模型、职级标准、培训资源、绩效记录、行业基准）。
3. 对标差距，生成分阶段目标、行动计划与评估指标。
4. 输出报告与图表（能力差距图、里程碑时间线）。

## 信息采集清单
- 基础信息：姓名、当前岗位、职级、年限、部门/区域。
- 目标意向：目标岗位/方向、期望时间、晋升/转岗诉求。
- 经验与成果：关键项目、职责范围、量化产出、影响面。
- 能力评估：能力项、当前等级、目标等级、证据或评价来源。
- 学习与认证：课程、证书、技能栈变化。
- 约束条件：时间投入、资源限制、岗位机会。

## 可检索数据源
- 岗位能力模型与职级标准（内部知识库/文档系统）。
- 绩效记录/OKR/360 反馈（数据库或人力系统）。
- 培训课程与学习路径（培训平台/LMS）。
- 行业能力基准与岗位画像（外部知识库）。
- 项目资源池与导师列表（内部项目库/人才库）。

## 输出结构（报告）
- 角色画像与目标画像
- 现状评估与差距分析（能力矩阵）
- 成长路径规划（短/中/长期里程碑）
- 行动建议（学习、项目、导师、反馈机制）
- 指标与评估节奏（里程碑验收、KPI/OKR）
- 风险与注意事项
- 数据来源与假设

## 图表要求
- 必出：能力差距图（当前 vs 目标）。
- 可选：里程碑时间线或路径流程图。
- 默认使用脚本生成 SVG；无法运行脚本时输出 Mermaid 图表。

## 脚本用法
```bash
python scripts/growth_path_planner.py \
  --input examples/growth_path_input.json \
  --output-md examples/growth_path_report.md \
  --output-svg examples/growth_path_chart.svg
```

## 输出说明
- `growth_path_report.md`：成长路径规划报告（含里程碑与行动建议）。
- `growth_path_chart.svg`：能力差距图（当前 vs 目标）。

## 输出注意
- 标注数据来源与假设，避免凭空结论。
- 明确关键差距与优先级，提出可执行的行动项与衡量指标。
- 输出图表与正文结论一致，避免指标口径不一致。
