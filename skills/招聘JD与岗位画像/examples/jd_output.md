# 资深后端工程师

- 部门：平台研发部
- 地点：上海
- 类型：全职
- 汇报对象：技术负责人

## 岗位职责
- 设计与实现核心服务，确保高并发下的稳定性
- 推动服务标准化与工程效率提升
- 与产品和运维团队协作落地需求

## 任职要求
- 5 年以上后端开发经验
- 熟悉 Rust 或 Go 生态
- 具备分布式系统设计经验

## 加分项
- 有云原生或 K8s 实战经验
- 了解 SRE 体系与可观测性建设

## 关键技能
- Rust
- PostgreSQL
- 分布式系统
- 性能优化

## 福利与发展
- 年度体检
- 培训预算
- 带薪年假
