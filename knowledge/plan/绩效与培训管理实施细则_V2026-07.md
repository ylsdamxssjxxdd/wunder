# 绩效与培训管理实施细则（V2026-07）

文档编号：`HR-PERFORMANCE-TRAINING-2026-07`  
生效日期：`2026-01-01`  

## 第1章 总则

### 1.1 目的
本细则旨在规范公司绩效管理和培训管理，建立科学、有效的绩效管理体系和培训发展体系，充分调动员工的积极性和创造性，提升员工能力素质，促进员工与公司共同发展，支撑公司战略目标实现。

### 1.2 适用范围
本细则适用于公司全体正式员工。劳务派遣员工、外包人员、实习人员参照执行。

### 1.3 基本原则
1. **战略导向原则**：绩效管理和培训管理必须服务于公司战略目标，支撑公司业务发展。
2. **客观公正原则**：绩效管理和培训管理必须客观公正，确保评价结果和培训效果真实可靠。
3. **持续改进原则**：绩效管理和培训管理必须持续改进，不断优化完善。
4. **员工参与原则**：绩效管理和培训管理必须充分调动员工的积极性，让员工主动参与。
5. **能力提升原则**：培训管理必须以能力提升为导向，注重员工能力培养和职业发展。
6. **绩效导向原则**：绩效管理必须以绩效为导向，注重工作成果和贡献。

## 第2章 绩效管理

### 2.1 绩效管理体系
公司建立完善的绩效管理体系，包括：
1. **绩效计划**：制定绩效目标和绩效计划。
2. **绩效辅导**：日常绩效辅导和沟通。
3. **绩效评估**：定期绩效评估。
4. **绩效反馈**：绩效反馈和面谈。
5. **绩效改进**：绩效改进和提升。
6. **绩效应用**：绩效结果应用。

### 2.2 绩效周期
绩效周期分为：
1. **年度绩效**：每年1月1日至12月31日。
2. **半年度绩效**：每年1月1日至6月30日，7月1日至12月31日。
3. **季度绩效**：每季度1日至季度最后1日。
4. **月度绩效**：每月1日至当月最后1日。

### 2.3 绩效指标
绩效指标分为：
1. **关键绩效指标（KPI）**：衡量关键成果的指标，权重60-80%。
2. **工作目标（GS）**：衡量工作任务完成情况的指标，权重20-40%。
3. **能力态度**：衡量员工能力和工作态度的指标，作为参考。

### 2.4 绩效计划制定
#### 2.4.1 绩效计划制定原则
绩效计划制定遵循以下原则：
1. **SMART原则**：
   - Specific（具体的）
   - Measurable（可衡量的）
   - Achievable（可实现的）
   - Relevant（相关的）
   - Time-bound（有时限的）
2. **上下结合原则**：上级与下级共同制定绩效计划。
3. **平衡原则**：短期目标与长期目标平衡，财务指标与非财务指标平衡。

#### 2.4.2 绩效计划制定流程
1. **绩效目标分解**：公司目标分解到部门，部门目标分解到个人。
2. **绩效计划沟通**：上级与下级沟通绩效目标和计划。
3. **绩效计划确认**：双方确认绩效计划，签字确认。
4. **绩效计划备案**：绩效计划报HR部门备案。

### 2.5 绩效辅导
#### 2.5.1 绩效辅导内容
绩效辅导包括：
1. **目标辅导**：帮助员工理解绩效目标。
2. **方法辅导**：指导员工工作方法和技能。
3. **资源支持**：为员工提供必要的资源支持。
4. **进度跟踪**：跟踪员工工作进度。
5. **问题解决**：帮助员工解决工作中的问题。

#### 2.5.2 绩效辅导方式
绩效辅导方式包括：
1. **日常沟通**：日常工作中的沟通交流。
2. **定期会议**：定期召开绩效沟通会议。
3. **一对一面谈**：一对一的绩效辅导面谈。
4. **书面反馈**：通过书面形式进行绩效反馈。

#### 2.5.3 绩效辅导频率
1. **日常沟通**：随时进行。
2. **定期会议**：每周或每月一次。
3. **一对一面谈**：每月至少一次。

### 2.6 绩效评估
#### 2.6.1 绩效评估时间
1. **月度绩效评估**：次月5日前完成。
2. **季度绩效评估**：次季度首月10日前完成。
3. **半年度绩效评估**：7月15日前和次年1月15日前完成。
4. **年度绩效评估**：次年1月20日前完成。

#### 2.6.2 绩效评估流程
1. **自我评估**：员工进行自我评估。
2. **上级评估**：上级对下级进行评估。
3. **同事评估**：同事之间相互评估（必要时）。
4. **下级评估**：下级对上级进行评估（必要时）。
5. **客户评估**：客户对员工进行评估（必要时）。
6. **绩效沟通**：上级与下级进行绩效沟通。
7. **绩效确认**：双方确认绩效评估结果。
8. **绩效备案**：绩效评估结果报HR部门备案。

#### 2.6.3 绩效等级
绩效等级分为：
| 绩效等级 | 定义 | 分布比例 | 薪酬系数 |
|---------|------|---------|---------|
| S | 卓越，远超预期 | 5-10% | 1.5-2.0 |
| A | 优秀，超出预期 | 15-20% | 1.2-1.5 |
| B | 良好，符合预期 | 50-60% | 1.0-1.2 |
| C | 待改进，部分符合预期 | 10-15% | 0.8-1.0 |
| D | 不合格，不符合预期 | 5%以下 | 0.5-0.8 |

### 2.7 绩效反馈
#### 2.7.1 绩效反馈内容
绩效反馈包括：
1. **绩效结果反馈**：反馈绩效评估结果。
2. **优点肯定**：肯定员工的优点和成绩。
3. **不足指出**：指出员工的不足和需要改进的地方。
4. **改进建议**：提出改进建议和措施。
5. **下期目标**：讨论下期绩效目标。

#### 2.7.2 绩效反馈面谈
绩效反馈面谈要求：
1. **面谈准备**：
   - 上级准备：收集绩效数据，准备面谈提纲。
   - 员工准备：准备自我总结，思考问题和建议。
2. **面谈过程**：
   - 营造轻松氛围。
   - 肯定成绩，增强信心。
   - 客观分析，指出不足。
   - 共同讨论，制定改进计划。
   - 明确目标，达成共识。
3. **面谈时间**：
   - 一般30-60分钟。
   - 选择安静、不受干扰的环境。
4. **面谈记录**：
   - 做好面谈记录。
   - 双方签字确认。

### 2.8 绩效改进
#### 2.8.1 绩效改进计划
绩效为C或D的员工，必须制定绩效改进计划：
1. **绩效改进目标**：明确绩效改进目标。
2. **绩效改进措施**：制定具体的改进措施。
3. **绩效改进时间**：明确改进时间进度。
4. **绩效改进资源**：明确所需资源支持。
5. **绩效改进评估**：明确评估标准和时间。

#### 2.8.2 绩效改进周期
绩效改进周期一般为1-3个月，特殊情况可以延长。

#### 2.8.3 绩效改进结果
绩效改进期满后，进行绩效评估：
1. **改进达标**：绩效达到B及以上，结束绩效改进。
2. **改进未达标**：
   - 延长绩效改进周期。
   - 调整岗位。
   - 培训学习。
   - 解除劳动合同。

### 2.9 绩效应用
#### 2.9.1 薪酬分配
1. **绩效奖金**：根据绩效等级发放绩效奖金。
2. **薪酬调整**：根据绩效等级进行薪酬调整。
3. **年终奖**：根据年度绩效等级发放年终奖。

#### 2.9.2 职业发展
1. **晋升**：绩效优秀的员工优先晋升。
2. **岗位调整**：根据绩效情况进行岗位调整。
3. **培训发展**：绩效优秀的员工优先参加培训。

#### 2.9.3 评优评先
1. **评优评先**：绩效优秀的员工优先参评各类荣誉。
2. **表彰奖励**：绩效优秀的员工优先获得表彰奖励。

#### 2.9.4 劳动合同管理
1. **续签**：绩效优秀的员工优先续签劳动合同。
2. **解除**：绩效不合格的员工，按规定解除劳动合同。

## 第3章 培训管理

### 3.1 培训体系
公司建立完善的培训体系，包括：
1. **培训需求分析**：分析培训需求。
2. **培训计划制定**：制定培训计划。
3. **培训实施**：组织实施培训。
4. **培训效果评估**：评估培训效果。
5. **培训档案管理**：管理培训档案。

### 3.2 培训分类
培训分为以下类别：
1. **按培训对象分**：
   - 新员工培训
   - 在职员工培训
   - 管理者培训
   - 专业技术人员培训
   - 营销人员培训
2. **按培训内容分**：
   - 入职培训
   - 岗位技能培训
   - 管理能力培训
   - 专业技术培训
   - 职业素养培训
   - 企业文化培训
3. **按培训方式分**：
   - 内部培训
   - 外部培训
   - 在线培训
   - 在岗培训
   - 脱岗培训
4. **按培训时间分**：
   - 短期培训
   - 中期培训
   - 长期培训

### 3.3 培训需求分析
#### 3.3.1 培训需求分析内容
培训需求分析包括：
1. **组织层面**：
   - 公司战略目标分析。
   - 公司经营计划分析。
   - 公司人力资源规划分析。
2. **岗位层面**：
   - 岗位职责分析。
   - 岗位任职资格分析。
   - 岗位绩效要求分析。
3. **个人层面**：
   - 员工能力现状分析。
   - 员工绩效现状分析。
   - 员工职业发展需求分析。

#### 3.3.2 培训需求分析方法
培训需求分析方法包括：
1. **问卷调查法**：通过问卷调查了解培训需求。
2. **访谈法**：通过访谈了解培训需求。
3. **观察法**：通过观察了解培训需求。
4. **绩效分析法**：通过绩效分析了解培训需求。
5. **能力测评法**：通过能力测评了解培训需求。

#### 3.3.3 培训需求分析流程
1. **制定培训需求分析计划**。
2. **收集培训需求信息**。
3. **分析培训需求信息**。
4. **形成培训需求分析报告**。
5. **审批培训需求分析报告**。

### 3.4 培训计划制定
#### 3.4.1 培训计划内容
培训计划包括：
1. **培训目标**：明确培训目标。
2. **培训对象**：明确培训对象。
3. **培训内容**：明确培训内容。
4. **培训方式**：明确培训方式。
5. **培训时间**：明确培训时间。
6. **培训地点**：明确培训地点。
7. **培训讲师**：明确培训讲师。
8. **培训预算**：明确培训预算。
9. **培训评估**：明确培训评估方式。

#### 3.4.2 培训计划制定流程
1. **各部门提出培训需求**。
2. **HR部门汇总培训需求**。
3. **HR部门制定培训计划**。
4. **征求各部门意见**。
5. **培训计划审批**。
6. **发布培训计划**。

### 3.5 培训实施
#### 3.5.1 培训准备
1. **确定培训讲师**：
   - 内部讲师：选拔、培养内部讲师。
   - 外部讲师：选择、评估外部讲师。
2. **准备培训材料**：
   - 培训教材。
   - 培训课件。
   - 培训案例。
   - 其他培训材料。
3. **布置培训场地**：
   - 选择培训场地。
   - 布置培训场地。
   - 准备培训设备。
4. **通知培训对象**：
   - 发布培训通知。
   - 确认参训人员。

#### 3.5.2 培训执行
1. **培训签到**：参训人员签到。
2. **培训开场**：培训开场介绍。
3. **培训授课**：培训讲师授课。
4. **培训互动**：培训互动交流。
5. **培训总结**：培训总结。
6. **培训考试**：必要时进行培训考试。

#### 3.5.3 培训纪律
1. 按时参加培训，不迟到、不早退、不无故缺席。
2. 培训期间保持安静，手机调至静音或震动状态。
3. 认真听讲，积极参与培训互动。
4. 遵守培训场地管理规定。
5. 爱护培训设施和培训材料。

### 3.6 培训效果评估
#### 3.6.1 培训评估层次
培训评估分为四个层次：
1. **反应层评估**：评估学员对培训的满意程度。
2. **学习层评估**：评估学员对知识、技能的掌握程度。
3. **行为层评估**：评估学员培训后行为的改变程度。
4. **结果层评估**：评估培训对组织绩效的影响程度。

#### 3.6.2 培训评估方法
培训评估方法包括：
1. **问卷调查法**：通过问卷调查评估培训效果。
2. **考试法**：通过考试评估学习效果。
3. **观察法**：通过观察评估行为改变。
4. **访谈法**：通过访谈评估培训效果。
5. **绩效分析法**：通过绩效分析评估结果层效果。

#### 3.6.3 培训评估流程
1. **制定培训评估计划**。
2. **收集培训评估数据**。
3. **分析培训评估数据**。
4. **形成培训评估报告**。
5. **反馈培训评估结果**。
6. **改进培训工作**。

### 3.7 培训档案管理
#### 3.7.1 培训档案内容
培训档案包括：
1. **员工培训档案**：
   - 员工培训记录表。
   - 培训考试成绩。
   - 培训证书。
   - 其他培训材料。
2. **培训管理档案**：
   - 培训需求分析报告。
   - 培训计划。
   - 培训实施记录。
   - 培训评估报告。
   - 其他培训管理材料。

#### 3.7.2 培训档案管理要求
1. 培训档案由HR部门统一管理。
2. 培训档案永久保存。
3. 培训档案查阅需按规定办理审批手续。
