# 知识产权与成果管理办法

# 第一章 总则

## 第一条 目的与依据

为加强公司知识产权与成果的规范化管理，保护公司核心智力成果，激励员工创新积极性，提升公司核心竞争力，防范知识产权侵权与流失风险，依据《中华人民共和国专利法》《中华人民共和国著作权法》《中华人民共和国商标法》及《中华人民共和国反不正当竞争法》等相关法律法规，结合公司研发、生产、经营实际情况，特制定本办法。

本办法旨在通过明确知识产权权属、规范成果申报与管理流程、建立科学激励与风险防控机制，实现知识产权“创造有激励、申报有规范、使用有约束、保护有保障”的管理目标，促进知识产权成果的转化与应用，支撑公司战略发展。

## 第二条 适用范围

本办法适用于公司全体劳动合同制员工、劳务派遣员工、实习生及外部合作单位（含顾问、合作研发机构等）在为公司提供服务过程中产生的各类知识产权与成果管理。涵盖的知识产权与成果类型包括：

1. 发明创造类：包括发明、实用新型、外观设计等专利成果；

2. 软件与数据类：包括计算机软件、算法模型、数据库、数据集合等软件类成果；

3. 作品与标识类：包括技术文档、图文作品、宣传材料、商标、商号、域名等标识与作品类成果；

4. 技术秘密类：包括未公开的技术方案、工艺配方、技术诀窍、研发数据等技术秘密成果；

5. 其他类：包括商业秘密、集成电路布图设计、植物新品种等符合法律法规规定的知识产权成果。

外部合作单位产生的与公司业务相关的知识产权与成果，需在合作协议中明确适用本办法相关条款，或另行签订专项管理协议。

## 第三条 基本原则

1. **战略导向原则**：知识产权与成果管理需紧密围绕公司核心业务与发展战略，优先保障核心技术、重点产品及新兴业务领域的知识产权布局与保护。

2. **权属清晰原则**：明确各类知识产权与成果的权属归属，通过协议约定、流程规范等方式，确保权属无争议、可追溯，避免权属纠纷。

3. **激励创新原则**：建立多层次、差异化的奖励机制，将成果创造、申报、转化与员工及团队的薪酬、晋升、荣誉等直接挂钩，激发创新活力。

4. **规范使用原则**：严格规范知识产权与成果的内部使用、外部授权及转化流程，强化全流程管控，防范滥用与流失风险。

5. **主动保护原则**：建立“预防为主、快速响应”的知识产权保护机制，主动开展权属登记、侵权监测与维权应对，保障公司合法权益。

6. **协同管理原则**：明确法务部、技术部、人力资源部、财务部及各业务部门的管理职责，形成“分工明确、协同高效”的管理体系。

## 第四条 管理机构与职责

公司建立“知识产权领导小组统筹、法务部牵头、各部门协同”的管理架构，明确各主体职责如下：

1. **知识产权领导小组**：由公司CEO担任组长，分管技术、法务的高管担任副组长，成员包括技术部、法务部、人力资源部、财务部及各业务部门负责人。主要职责：审定知识产权战略规划与年度计划；审批重大知识产权决策（如核心专利申报、对外授权、维权重大事项等）；协调解决知识产权管理中的重大问题。

2. **法务部（知识产权管理办公室）**：作为知识产权与成果管理的日常执行部门，设专职知识产权专员。主要职责：制定与修订知识产权管理制度；负责知识产权权属登记、申请、维护与档案管理；组织成果评审与知识资产库建设；审核知识产权对外授权、转让及合作协议；牵头开展侵权监测、维权应对与纠纷处理；开展知识产权培训与宣传。

3. **技术部（含研发中心、项目管理部）**：作为成果创造与申报的核心部门。主要职责：组织研发团队开展成果梳理与申报；提供成果相关的技术资料与说明；参与成果评审与价值评估；配合法务部开展专利布局、技术秘密保护及侵权排查；推动知识产权成果的内部转化与应用。

4. **人力资源部**：负责将知识产权成果纳入员工与团队考核激励体系；组织新员工签署知识产权归属与保密协议；落实知识产权相关奖励的核算与发放；开展员工知识产权意识培训。

5. **财务部**：负责知识产权申请、维护、维权等费用的预算与拨付；核算知识产权成果转化收益及奖励金额；对知识产权资产进行会计核算与价值管理。

6. **各业务部门**：负责本部门业务相关知识产权与成果的初步梳理与申报；规范本部门知识产权的使用与保密；发现侵权或被侵权线索及时上报；配合法务部开展相关工作。

# 第二章 知识产权权属界定

## 第五条 职务成果界定标准

员工在任职期间及离职后1年内（竞业限制期内）产生的下列成果，均界定为职务成果，知识产权归公司所有：

1. 直接利用公司的物质技术条件（如资金、设备、场地、技术资料、研发数据等）完成的与公司业务相关的发明创造、软件、技术方案等；

2. 履行本职工作或公司分配的专项任务所完成的成果；

3. 退职、退休或调动工作后1年内，与原任职岗位的职责或公司分配的任务相关的成果；

4. 员工在公司组织的培训、项目合作、学术交流等活动中，基于公司资源或业务需求产生的成果；

5. 双方在劳动合同或专项协议中明确约定为职务成果的其他情形。

职务成果的完成人享有署名权、获得奖励权，但不得自行主张知识产权权属，不得擅自转让、许可或公开。

## 第六条 非职务成果界定标准

员工在任职期间产生的下列成果，界定为非职务成果，知识产权归个人所有，但需向公司履行备案手续：

1. 未利用公司任何物质技术条件，与本职工作及公司业务完全无关的成果；

2. 在非工作时间独立完成，且未借鉴公司任何技术资料、未参与公司相关研发项目的成果；

3. 员工在入职前已完成并提供有效证明（如专利申请受理通知书、作品发表证明等）的成果。

员工主张非职务成果的，需在成果形成后15个工作日内，向法务部提交《非职务成果备案申请表》及相关证明材料，经法务部审核确认后备案。未按规定备案的，视为职务成果。

## 第七条 权属协议管理

为确保权属清晰可追溯，公司实行“全员协议覆盖、专项场景补充”的协议管理机制：

1. **入职协议签署**：新员工入职当日，需与公司签署《知识产权归属与保密协议》，明确职务成果与非职务成果的界定标准、权属归属及保密义务。未签署协议的，不予办理入职手续。

2. **专项协议补充**：对于核心研发岗位、技术秘密岗位员工，及参与重大研发项目、涉密项目的人员，需额外签署《专项知识产权协议》，细化成果归属、保密责任及竞业限制相关条款。

3. **外部合作协议**：与外部合作单位（含顾问、合作研发机构）开展合作时，需在合作协议中明确知识产权归属条款，优先约定成果归公司所有；若约定共有，需明确公司的使用、转让、许可等优先权。

4. **协议存档管理**：《知识产权归属与保密协议》及相关专项协议由人力资源部与法务部各留存一份，存档期限为员工离职后5年；外部合作协议由法务部存档，存档期限为合作结束后10年。

## 第八条 权属争议处理

发生知识产权权属争议时，按以下流程处理：

1. **申请发起**：争议方（员工或部门）可向法务部提交《知识产权权属争议申请书》，说明争议事实、理由及相关证据。

2. **调查核实**：法务部在收到申请后10个工作日内，联合技术部、人力资源部及相关部门开展调查，收集成果形成过程中的资金投入、设备使用、工作记录等证据。

3. **争议评审**：由知识产权领导小组组织技术专家、法务专家及第三方专业机构（必要时）组成评审小组，结合调查结果及相关法律法规，出具《权属争议评审意见》。

4. **结果执行**：评审意见经知识产权领导小组审批后生效，争议方需按意见执行。对评审结果有异议的，可在收到结果后5个工作日内申请复核，复核结果为最终结论。

权属争议处理期间，争议成果暂按职务成果管理，未经法务部批准，任何一方不得擅自处置。

# 第三章 成果申报与管理

## 第九条 成果申报范围与时限

各部门及员工在成果形成后，需按以下要求及时申报：

1. **申报范围**：包括但不限于：已完成的发明创造、实用新型技术方案；已开发完成的软件程序及相关文档；已形成的技术秘密、工艺配方；已完成的图文作品、商标设计；已实现初步商业化的技术成果等。

2. **申报时限**：研发/项目组在成果形成后10个工作日内，需完成成果梳理并提交申报；个人独立完成的职务成果，需在成果形成后15个工作日内提交申报；外部合作单位按合作协议约定的时限提交成果申报材料。

3. **延迟申报处理**：因故意或重大过失延迟申报，导致成果丧失新颖性、创造性或被他人抢先申请的，追究相关责任人的责任；因客观原因无法按时申报的，需提前向法务部提交《成果申报延期申请》，经批准后可延期，延期最长不超过10个工作日。

## 第十条 申报材料与流程

成果申报实行“部门初审、法务部审核、联合评审”的流程，具体要求如下：

1. **材料准备**：申报单位或个人需提交《成果登记表》（明确成果名称、类型、完成人、形成时间、技术要点、应用价值等），并附相关证明材料（如技术方案、软件代码、设计图纸、测试报告等）。

2. **部门初审**：申报材料经部门负责人审核，确认成果真实性、完整性及与业务的关联性后，签字提交至法务部。核心研发成果需同时经技术部负责人初审。

3. **法务部审核**：法务部对申报材料的规范性、权属清晰度进行审核，对材料不全的，通知申报方在5个工作日内补充；对符合要求的，组织联合评审。

4. **联合评审**：由法务部牵头，组织技术委员会（由公司技术专家组成）开展联合评审，评审内容包括成果的创新性、实用性、知识产权保护价值及转化潜力，形成《成果评审报告》，明确成果分类（核心成果、重要成果、一般成果）及保护建议（如专利申请、软著登记、技术秘密保护等）。

5. **结果反馈**：法务部在评审结束后5个工作日内，将评审结果及保护建议反馈至申报部门及个人，并同步至知识产权领导小组备案。

## 第十一条 知识资产库建设与管理

为实现成果的系统化管理与高效利用，公司建立“线上知识资产库”，实行“统一收录、分类管理、授权使用”的机制：

1. **资产收录**：经联合评审后的所有成果，由法务部在3个工作日内完成知识资产库录入，明确资产编号、权属信息、保护方式、使用权限等核心信息，并附完整申报与评审材料。

2. **分类管理**：知识资产库按成果类型分为专利库、软件库、技术秘密库、作品库等子库，每个子库按“核心-重要-一般”等级分类管理，核心成果设置专项保护标识。

3. **动态更新**：成果的保护状态（如专利申请进展、软著登记结果）、转化情况、维护记录等信息，由法务部每月更新至知识资产库，确保资产信息实时准确。

4. **授权使用**：知识资产库对内部员工开放查询权限，核心成果的查看与使用需经法务部及相关部门负责人授权；资产库数据作为AI人才管理系统、研发决策系统的权威数据来源，为技术研发、人才评价等提供数据支持。

5. **安全管理**：知识资产库由IT部门负责技术安全保障，设置访问日志与操作记录，严禁擅自下载、复制或外传库内核心成果信息，法务部定期开展安全检查。

## 第十二条 成果维护与终止

公司对已收录的知识产权成果实行“全生命周期维护”：

1. **定期维护**：法务部每季度对知识资产库中的成果进行梳理，对专利、软著等需续展的知识产权，提前6个月提醒财务部准备费用，按时完成续展；对技术秘密成果，联合技术部评估保密状态，更新保密措施。

2. **价值评估**：每年年底，由法务部、技术部、财务部联合开展成果价值评估，对无应用价值、保护成本过高或已公开的成果，提出终止保护建议，报知识产权领导小组审批后，在知识资产库中标记并说明原因。

3. **档案留存**：终止保护的成果档案，由法务部继续留存5年，确保追溯可查。

# 第四章 成果奖励与激励

## 第十三条 奖励原则与分类

成果奖励实行“按价值分配、按贡献激励”的原则，分为“申报奖励、授权奖励、转化奖励”三类，兼顾团队与个人激励，确保奖励公开、公平、及时。

奖励标准由人力资源部联合财务部、法务部、技术部根据成果类型、价值及公司经营情况制定《知识产权成果奖励标准》，经知识产权领导小组审批后发布实施，并根据年度情况动态调整。

## 第十四条 具体奖励措施

1. **申报奖励**：对按要求及时完成成果申报，且材料完整、质量较高的团队或个人，给予一次性申报奖励。其中，核心成果申报奖励500-1000元/项，重要成果300-500元/项，一般成果100-300元/项，由团队负责人统筹分配。

2. **授权奖励**：成果获得知识产权授权的，给予授权奖励：
        发明专利授权：团队奖励20000-50000元/项，核心完成人（前3名）额外奖励5000-10000元/人；

3. 实用新型专利授权：团队奖励5000-10000元/项，核心完成人额外奖励1000-3000元/人；

4. 软件著作权登记：团队奖励2000-5000元/项，核心完成人额外奖励500-1000元/人；

5. 商标注册成功：根据商标重要性，给予申请团队1000-3000元/项奖励。

6. **转化奖励**：成果实现商业化转化或内部应用产生显著价值的，给予转化奖励：
        外部转化：成果通过转让、许可等方式实现外部收益的，按年度实际收益的10%-20%给予团队奖励，其中核心完成人分配比例不低于奖励总额的40%；

7. 内部应用：成果应用于公司产品或业务，使产品成本降低、效率提升或销售额增长的，按年度新增效益的5%-10%给予团队奖励，具体金额由财务部核算；

8. 股权与署名激励：对为公司带来重大经济效益的核心成果（如核心专利支撑主力产品上市），除现金奖励外，可给予核心完成人股权期权激励；在学术成果、行业奖项申报中，优先保障成果完成人的署名权。

## 第十五条 奖励申报与发放流程

1. **奖励申报**：符合奖励条件的团队或个人，由团队负责人或成果完成人向人力资源部提交《成果奖励申请表》，并附相关证明材料（如专利证书、软著登记证、转化协议、收益证明等）。

2. **审核核算**：人力资源部联合法务部审核奖励资格与材料真实性，财务部核算奖励金额，形成《成果奖励审核表》。

3. **审批发放**：审核表经知识产权领导小组审批后，人力资源部在当月薪酬发放时将奖励足额发放至相关人员，并在公司内部公示奖励情况（保护个人隐私的信息除外）。

4. **记录存档**：奖励发放后，人力资源部将奖励信息记入员工绩效档案，法务部同步更新知识资产库中的成果激励记录，相关材料由人力资源部与法务部共同存档。

## 第十六条 其他激励措施

除物质奖励外，公司通过以下方式强化激励效果：

## 第二十条 侵权监测与应对

公司建立“主动监测、快速响应、依法维权”的知识产权保护机制：

8. **后续跟进**：维权案件结束后，法务部需及时将处理结果归档，并针对侵权风险点，优化公司知识产权保护措施，加强相关领域的监测力度。

        擅自对外授权、转让核心成果，或擅自将软件开源、上传公共仓库的；

5. 虚报成果转化收益或奖励申请材料，骗取奖励的；
