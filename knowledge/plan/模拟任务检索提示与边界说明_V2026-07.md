# 1. 目的

本说明用于约束模拟任务中的检索行为，避免仅凭单条片段做结论。

# 2. 推荐检索顺序

1. 先检索制度汇编总文档中的场景章节；
2. 再检索绩效/培训、纪律、例外细则；
3. 最后回到数据库字段做事实核验。

# 3. 结论生成要求

- 至少引用 2 类不同来源条款（主规则 + 配套细则）；
- 结论必须可映射到 RuleID；
- 若条款存在多个版本，需说明使用版本。
