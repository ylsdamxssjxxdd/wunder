# 考勤与休假管理实施细则（V2026-07）

文档编号：`HR-ATTENDANCE-LEAVE-2026-07`  
生效日期：`2026-01-01`  

## 第1章 总则

### 1.1 目的
本细则旨在规范公司员工考勤与休假管理，建立科学、规范的考勤与休假制度，维护公司正常的工作秩序，保障员工的休息休假权利，提高工作效率，促进公司持续健康发展。

### 1.2 适用范围
本细则适用于公司全体正式员工。劳务派遣员工、外包人员、实习人员参照执行。

### 1.3 基本原则
1. **依法合规原则**：考勤与休假管理必须符合国家法律法规和公司规章制度。
2. **公平公正原则**：考勤与休假管理必须公平公正，对所有员工一视同仁。
3. **效率优先原则**：考勤与休假管理必须高效，简化流程，提高效率。
4. **人性化原则**：考勤与休假管理必须体现人性化，兼顾公司和员工利益。
5. **严肃性原则**：考勤纪律必须严肃，确保制度执行到位。

## 第2章 工作时间

### 2.1 标准工作时间
公司实行标准工作时间制度：
1. **工作日**：每周一至周五。
2. **工作时间**：
   - 上午：9:00-12:00
   - 下午：13:00-18:00
3. **每日工作时间**：8小时。
4. **每周工作时间**：40小时。

### 2.2 弹性工作时间
公司对部分岗位实行弹性工作时间制度：
1. **适用范围**：
   - 部分管理岗位。
   - 部分专业技术岗位。
   - 其他适合实行弹性工作时间的岗位。
2. **弹性工作时间要求**：
   - 每日工作时间不少于8小时。
   - 每周工作时间不少于40小时。
   - 核心工作时间（10:00-16:00）必须在岗。
   - 具体工作时间由员工与部门负责人协商确定。
3. **弹性工作时间审批**：
   - 员工填写《弹性工作时间申请表》。
   - 部门负责人审核。
   - HR部门审批。
   - 备案执行。

### 2.3 不定时工作制
公司对部分岗位实行不定时工作制：
1. **适用范围**：
   - 高级管理人员。
   - 部分外勤人员。
   - 部分特殊岗位人员。
2. **不定时工作制要求**：
   - 以完成工作任务为目标，不固定工作时间。
   - 确保工作任务按时完成。
   - 保持通讯畅通，便于工作联系。
3. **不定时工作制审批**：
   - 按照国家规定，报劳动行政部门审批。
   - 公司内部审批备案。

### 2.4 综合计算工时工作制
公司对部分岗位实行综合计算工时工作制：
1. **适用范围**：
   - 部分生产岗位。
   - 部分服务岗位。
   - 其他适合实行综合计算工时工作制的岗位。
2. **综合计算工时工作制要求**：
   - 以周、月、季、年为周期综合计算工作时间。
   - 平均日工作时间和平均周工作时间不超过法定标准工作时间。
   - 确保员工休息休假权利。
3. **综合计算工时工作制审批**：
   - 按照国家规定，报劳动行政部门审批。
   - 公司内部审批备案。

## 第3章 考勤管理

### 3.1 考勤方式
公司采用多种考勤方式：
1. **打卡考勤**：
   - 指纹打卡。
   - 人脸识别打卡。
   - 打卡机打卡。
   - 手机APP打卡。
2. **签到考勤**：
   - 纸质签到。
   - 电子签到。
3. **其他考勤方式**：
   - 部门负责人考勤。
   - 项目负责人考勤。

### 3.2 考勤时间
1. **上班打卡时间**：
   - 标准工作时间：9:00以前。
   - 弹性工作时间：根据协商确定的时间。
2. **下班打卡时间**：
   - 标准工作时间：18:00以后。
   - 弹性工作时间：根据协商确定的时间。
3. **忘记打卡**：
   - 员工忘记打卡，应在24小时内填写《忘记打卡说明表》。
   - 部门负责人审核确认。
   - HR部门备案。
   - 每月忘记打卡不得超过3次，超过3次按迟到或早退处理。

### 3.3 迟到与早退
1. **迟到**：
   - 晚于规定上班时间打卡为迟到。
   - 迟到10分钟以内：口头警告。
   - 迟到10-30分钟：书面警告，扣发绩效奖金。
   - 迟到30分钟以上：按旷工半天处理。
   - 月度累计迟到5次以上：按旷工1天处理。
2. **早退**：
   - 早于规定下班时间打卡为早退。
   - 早退10分钟以内：口头警告。
   - 早退10-30分钟：书面警告，扣发绩效奖金。
   - 早退30分钟以上：按旷工半天处理。
   - 月度累计早退5次以上：按旷工1天处理。

### 3.4 旷工
1. **旷工情形**：
   - 未请假或请假未批准不到岗。
   - 假期期满未续假或续假未批准不到岗。
   - 迟到、早退30分钟以上。
   - 其他视为旷工的情形。
2. **旷工处理**：
   - 旷工半天：扣发2天工资，书面警告。
   - 旷工1天：扣发3天工资，记过处分。
   - 连续旷工2天：扣发5天工资，记大过处分。
   - 连续旷工3天或年度累计旷工5天：解除劳动合同。

### 3.5 外出管理
1. **因公外出**：
   - 员工因公外出，应填写《外出登记表》。
   - 部门负责人审批。
   - 外出时间较长的，应进行打卡考勤。
2. **因私外出**：
   - 员工因私外出，应提前请假。
   - 按请假流程办理。

### 3.6 考勤统计
1. **考勤周期**：每月1日至当月最后1日。
2. **考勤统计**：
   - 部门负责人每月5日前将上月考勤情况报HR部门。
   - HR部门每月10日前完成上月考勤统计。
   - 考勤统计结果作为工资核算、绩效评估的重要依据。
3. **考勤核对**：
   - 员工对考勤统计结果有异议的，应在3个工作日内提出。
   - HR部门及时核实并调整。

## 第4章 休假管理

### 4.1 法定节假日
公司按照国家规定安排员工享受法定节假日：
1. **全体公民放假的节日**：
   - 新年，放假1天（1月1日）。
   - 春节，放假3天（农历正月初一、初二、初三）。
   - 清明节，放假1天（农历清明当日）。
   - 劳动节，放假1天（5月1日）。
   - 端午节，放假1天（农历端午当日）。
   - 中秋节，放假1天（农历中秋当日）。
   - 国庆节，放假3天（10月1日、2日、3日）。
2. **部分公民放假的节日及纪念日**：
   - 妇女节（3月8日），妇女放假半天。
   - 青年节（5月4日），14周岁以上的青年放假半天。
   - 儿童节（6月1日），不满14周岁的少年儿童放假1天。
   - 中国人民解放军建军纪念日（8月1日），现役军人放假半天。
3. **其他**：
   - 少数民族习惯的节日，按照各该民族习惯，规定放假日期。
   - 法定节假日适逢休息日，应当在工作日补假。

### 4.2 带薪年休假
1. **年休假天数**：
   - 员工累计工作已满1年不满10年的，年休假5天。
   - 已满10年不满20年的，年休假10天。
   - 已满20年的，年休假15天。
2. **年休假享受条件**：
   - 员工连续工作1年以上的，享受带薪年休假。
   - 员工有下列情形之一的，不享受当年的年休假：
     - 员工依法享受寒暑假，其休假天数多于年休假天数的。
     - 员工请事假累计20天以上且单位按照规定不扣工资的。
     - 累计工作满1年不满10年的员工，请病假累计2个月以上的。
     - 累计工作满10年不满20年的员工，请病假累计3个月以上的。
     - 累计工作满20年以上的员工，请病假累计4个月以上的。
3. **年休假安排**：
   - 公司根据生产、工作的具体情况，并考虑员工本人意愿，统筹安排员工年休假。
   - 年休假在1个年度内可以集中安排，也可以分段安排，一般不跨年度安排。
   - 公司因生产、工作特点确有必要跨年度安排员工年休假的，可以跨1个年度安排。
4. **年休假申请**：
   - 员工提前填写《年休假申请表》。
   - 部门负责人审核。
   - HR部门审批。
   - 备案执行。
5. **未休年休假处理**：
   - 公司确因工作需要不能安排员工休年休假的，经员工本人同意，可以不安排员工休年休假。
   - 对员工应休未休的年休假天数，公司应当按照该员工日工资收入的300%支付年休假工资报酬。

### 4.3 病假
1. **病假申请**：
   - 员工因病需要休假的，应提前填写《病假申请表》。
   - 急诊可以先电话请假，事后3个工作日内补办请假手续。
   - 提交医院出具的病假证明。
2. **病假审批**：
   - 病假3天以内：部门负责人审批。
   - 病假3-7天：部门负责人审核，业务分管领导审批。
   - 病假7天以上：部门负责人审核，业务分管领导审批，HR部门备案。
3. **病假待遇**：
   - 病假在6个月以内的，按以下标准支付病假工资：
     - 连续工龄不满2年的，按本人工资的60%计发。
     - 连续工龄满2年不满4年的，按本人工资的70%计发。
     - 连续工龄满4年不满6年的，按本人工资的80%计发。
     - 连续工龄满6年不满8年的，按本人工资的90%计发。
     - 连续工龄满8年及以上的，按本人工资的100%计发。
   - 病假超过6个月的，按以下标准支付疾病救济费：
     - 连续工龄不满1年的，按本人工资的40%计发。
     - 连续工龄满1年不满3年的，按本人工资的50%计发。
     - 连续工龄满3年及以上的，按本人工资的60%计发。
   - 病假工资和疾病救济费不得低于当地最低工资标准的80%。
4. **医疗期**：
   - 企业职工因患病或非因工负伤，需要停止工作医疗时，根据本人实际参加工作年限和在本单位工作年限，给予3个月到24个月的医疗期：
     - 实际工作年限10年以下的，在本单位工作年限5年以下的为3个月；5年以上的为6个月。
     - 实际工作年限10年以上的，在本单位工作年限5年以下的为6个月；5年以上10年以下的为9个月；10年以上15年以下的为12个月；15年以上20年以下的为18个月；20年以上的为24个月。
   - 医疗期3个月的按6个月内累计病休时间计算；6个月的按12个月内累计病休时间计算；9个月的按15个月内累计病休时间计算；12个月的按18个月内累计病休时间计算；18个月的按24个月内累计病休时间计算；24个月的按30个月内累计病休时间计算。

### 4.4 事假
1. **事假申请**：
   - 员工因事需要休假的，应提前填写《事假申请表》。
   - 紧急情况可以先电话请假，事后1个工作日内补办请假手续。
2. **事假审批**：
   - 事假1天以内：部门负责人审批。
   - 事假1-3天：部门负责人审核，业务分管领导审批。
   - 事假3天以上：部门负责人审核，业务分管领导审批，HR部门备案。
3. **事假待遇**：
   - 事假期间不发放工资。
   - 月度累计事假超过5天的，扣发当月绩效奖金。
   - 年度累计事假超过20天的，不享受当年年休假。

### 4.5 婚假
1. **婚假天数**：
   - 员工结婚，享受婚假3天。
   - 符合晚婚条件的，增加婚假天数（按地方规定执行）。
2. **婚假申请**：
   - 员工提前填写《婚假申请表》。
   - 提交结婚证明。
3. **婚假审批**：
   - 部门负责人审核。
   - HR部门审批。
4. **婚假待遇**：
   - 婚假期间，工资照发。
   - 婚假应当一次性连续使用，遇法定节假日顺延。

### 4.6 产假
1. **产假天数**：
   - 女职工生育享受98天产假，其中产前可以休假15天。
   - 难产的，增加产假15天。
   - 生育多胞胎的，每多生育1个婴儿，增加产假15天。
   - 符合地方规定的，增加产假天数（按地方规定执行）。
2. **产假申请**：
   - 员工提前填写《产假申请表》。
   - 提交生育证明。
3. **产假审批**：
   - 部门负责人审核。
   - HR部门审批。
4. **产假待遇**：
   - 产假期间，享受生育津贴。
   - 生育津贴按照员工所在用人单位上年度职工月平均工资计发。
   - 生育津贴低于员工产假前工资标准的，由用人单位补足差额。
   - 生育津贴高于员工产假前工资标准的，用人单位不得截留。

### 4.7 陪产假
1. **陪产假天数**：
   - 男职工在配偶生育时，享受陪产假（按地方规定执行，一般7-30天）。
2. **陪产假申请**：
   - 员工提前填写《陪产假申请表》。
   - 提交配偶生育证明。
3. **陪产假审批**：
   - 部门负责人审核。
   - HR部门审批。
4. **陪产假待遇**：
   - 陪产假期间，工资照发。

### 4.8 哺乳假
1. **哺乳假天数**：
   - 女职工哺乳未满1周岁婴儿的，每天安排1小时哺乳时间。
   - 生育多胞胎的，每多哺乳1个婴儿每天增加1小时哺乳时间。
2. **哺乳假申请**：
   - 员工填写《哺乳假申请表》。
   - 提交婴儿出生证明。
3. **哺乳假审批**：
   - 部门负责人审批。
4. **哺乳假待遇**：
   - 哺乳时间计入工作时间。

### 4.9 探亲假
1. **探亲假享受条件**：
   - 在公司工作满1年的员工，与配偶不住在一起，又不能在公休假日团聚的，可以享受探望配偶的待遇。
   - 与父亲、母亲都不住在一起，又不能在公休假日团聚的，可以享受探望父母的待遇。
2. **探亲假天数**：
   - 员工探望配偶的，每年给予一方探亲假一次，假期为30天。
   - 未婚员工探望父母，原则上每年给假一次，假期为20天；如果因为工作需要，公司当年不能给予假期，或者员工自愿两年探亲一次的，可以两年给假一次，假期为45天。
   - 已婚员工探望父母的，每4年给假一次，假期为20天。
   - 探亲假期是指员工与配偶、父、母团聚的时间，另外，根据实际需要给予路程假。上述假期均包括公休假日和法定节日在内。
3. **探亲假申请**：
   - 员工提前填写《探亲假申请表》。
4. **探亲假审批**：
   - 部门负责人审核。
   - HR部门审批。
5. **探亲假待遇**：
   - 员工在规定的探亲假期和路程假期内，按照本人的标准工资发给工资。
   - 员工探望配偶和未婚员工探望父母的往返路费，由公司负担。
   - 已婚员工探望父母的往返路费，在本人月标准工资30%以内的，由本人自理，超过部分由公司负担。

### 4.10 丧假
1. **丧假天数**：
   - 员工的直系亲属（父母、配偶、子女）死亡时，给予丧假3天。
   - 员工的其他亲属死亡时，根据实际情况给予丧假1-3天。
   - 需要到外地料理丧事的，根据路程远近，另给予路程假。
2. **丧假申请**：
   - 员工填写《丧假申请表》。
   - 提交亲属死亡证明。
3. **丧假审批**：
   - 部门负责人审批。
   - HR部门备案。
4. **丧假待遇**：
   - 丧假和路程假期间，工资照发。
   - 往返路费由员工自理。

### 4.11 工伤假
1. **工伤假享受条件**：
   - 员工因工作遭受事故伤害或者患职业病需要暂停工作接受工伤医疗的，享受工伤假。
2. **工伤假天数**：
   - 停工留薪期一般不超过12个月。
   - 伤情严重或者情况特殊，经设区的市级劳动能力鉴定委员会确认，可以适当延长，但延长不得超过12个月。
3. **工伤假待遇**：
   - 在停工留薪期内，原工资福利待遇不变，由公司按月支付。
   - 工伤职工评定伤残等级后，停发原待遇，按照有关规定享受伤残待遇。
   - 工伤职工在停工留薪期满后仍需治疗的，继续享受工伤医疗待遇。

## 第5章 请假流程

### 5.1 请假申请
1. 员工请假应提前填写《请假申请表》，说明请假类型、请假时间、请假事由等。
2. 按要求提交相关证明材料。
3. 提前请假时间要求：
   - 年休假：提前1周申请。
   - 病假：提前1天申请，急诊可事后补办。
   - 事假：提前1天申请，紧急情况可事后补办。
   - 婚假：提前2周申请。
   - 产假：提前1个月申请。
   - 陪产假：提前1周申请。
   - 探亲假：提前2周申请。
   - 丧假：及时申请。

### 5.2 请假审批
1. 员工填写《请假申请表》。
2. 部门负责人审核。
3. 按审批权限报相关领导审批。
4. HR部门备案。
5. 审批通过后，员工方可休假。

### 5.3 请假交接
1. 员工请假前，应做好工作交接。
2. 指定工作代理人，确保工作正常进行。
3. 填写《工作交接清单》，交接双方签字确认。

### 5.4 销假
1. 员工假期期满后，应及时到岗工作。
2. 到岗后1个工作日内到HR部门办理销假手续。
3. 如需续假，应提前办理续假手续。

## 第6章 加班管理

### 6.1 加班原则
1. 公司不提倡加班，鼓励员工在工作时间内完成工作任务。
2. 确因工作需要加班的，应严格控制加班时间。
3. 加班应事先审批，未经审批的加班不予认可。

### 6.2 加班申请
1. 员工加班应提前填写《加班申请表》，说明加班时间、加班事由等。
2. 部门负责人审核。
3. 按审批权限报相关领导审批。
4. 审批通过后，方可加班。

### 6.3 加班审批权限
1. 加班1-2小时：部门负责人审批。
2. 加班2-4小时：业务分管领导审批。
3. 加班4小时以上：CEO审批。

### 6.4 加班待遇
1. **调休**：
   - 员工加班可以安排调休。
   - 调休应在加班后6个月内使用。
   - 调休时间与加班时间相等。
2. **加班工资**：
   - 安排员工延长工作时间的，支付不低于工资的150%的工资报酬。
   - 休息日安排员工工作又不能安排补休的，支付不低于工资的200%的工资报酬。
   - 法定休假日安排员工工作的，支付不低于工资的300%的工资报酬。
