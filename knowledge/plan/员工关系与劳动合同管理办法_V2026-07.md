# 员工关系与劳动合同管理办法（V2026-07）

文档编号：`HR-EMPLOYEE-RELATIONS-2026-07`  
生效日期：`2026-01-01`  

## 第1章 总则

### 1.1 目的
本办法旨在规范公司员工关系与劳动合同管理，建立和谐稳定的劳动关系，保障公司和员工的合法权益，促进公司持续健康发展。

### 1.2 适用范围
本办法适用于公司全体正式员工。劳务派遣员工、外包人员参照执行。

### 1.3 基本原则
1. **依法合规原则**：员工关系与劳动合同管理必须符合国家法律法规和公司规章制度。
2. **公平公正原则**：员工关系与劳动合同管理必须公平公正，保障员工的合法权益。
3. **和谐稳定原则**：员工关系与劳动合同管理必须致力于建立和谐稳定的劳动关系。
4. **沟通协商原则**：处理员工关系问题必须坚持沟通协商，及时化解矛盾。
5. **预防为主原则**：员工关系管理必须坚持预防为主，及时发现和解决问题。

## 第2章 劳动合同管理

### 2.1 劳动合同订立
1. **订立原则**：
   - 合法原则
   - 公平原则
   - 平等自愿原则
   - 协商一致原则
   - 诚实信用原则
2. **订立时间**：
   - 新员工入职之日起1个月内订立书面劳动合同。
   - 已建立劳动关系，未同时订立书面劳动合同的，应当自用工之日起1个月内订立。
3. **劳动合同类型**：
   - 固定期限劳动合同
   - 无固定期限劳动合同
   - 以完成一定工作任务为期限的劳动合同
4. **劳动合同内容**：
   - 用人单位的名称、住所和法定代表人或者主要负责人。
   - 劳动者的姓名、住址和居民身份证或者其他有效身份证件号码。
   - 劳动合同期限。
   - 工作内容和工作地点。
   - 工作时间和休息休假。
   - 劳动报酬。
   - 社会保险。
   - 劳动保护、劳动条件和职业危害防护。
   - 法律、法规规定应当纳入劳动合同的其他事项。
   - 试用期、培训、保守秘密、补充保险和福利待遇等其他约定事项。

### 2.2 劳动合同履行
1. **公司义务**：
   - 按照劳动合同约定和国家规定，向员工及时足额支付劳动报酬。
   - 严格执行劳动定额标准，不得强迫或者变相强迫员工加班。
   - 安排员工加班的，按照国家有关规定向员工支付加班费。
   - 保护员工的生命安全和身体健康，为员工提供符合国家规定的劳动安全卫生条件和必要的劳动防护用品。
   - 依法建立和完善劳动规章制度，保障员工享有劳动权利、履行劳动义务。
2. **员工义务**：
   - 完成劳动任务，提高职业技能。
   - 执行劳动安全卫生规程，遵守劳动纪律和职业道德。
   - 遵守公司依法制定的规章制度。
   - 保守公司的商业秘密。
   - 履行劳动合同约定的其他义务。

### 2.3 劳动合同变更
1. **变更情形**：
   - 公司与员工协商一致，可以变更劳动合同约定的内容。
   - 劳动合同订立时所依据的客观情况发生重大变化，致使劳动合同无法履行，经公司与员工协商，可以变更劳动合同。
2. **变更程序**：
   - 一方提出变更劳动合同的建议。
   - 双方协商变更内容。
   - 达成一致意见后，签订书面变更协议。
   - 变更协议双方各执一份。
3. **变更要求**：
   - 变更劳动合同，应当采用书面形式。
   - 变更后的劳动合同文本由公司和员工各执一份。

### 2.4 劳动合同续签
1. **续签评估**：
   - 劳动合同期满前30天，用人部门对员工进行续签评估。
   - 评估内容包括工作表现、工作能力、绩效情况、岗位匹配度等。
2. **续签程序**：
   - 用人部门提出续签意见。
   - HR部门审核。
   - 按审批权限报相关领导审批。
   - 审批通过后，与员工协商续签事宜。
   - 协商一致后，签订书面劳动合同。
3. **无固定期限劳动合同**：
   - 员工在公司连续工作满10年的。
   - 公司初次实行劳动合同制度或者国有企业改制重新订立劳动合同时，员工在公司连续工作满10年且距法定退休年龄不足10年的。
   - 连续订立二次固定期限劳动合同，且员工没有《劳动合同法》第三十九条和第四十条第一项、第二项规定的情形，续订劳动合同的。

### 2.5 劳动合同解除
#### 2.5.1 协商解除
公司与员工协商一致，可以解除劳动合同。

#### 2.5.2 员工提前通知解除
员工提前30日以书面形式通知公司，可以解除劳动合同。员工在试用期内提前3日通知公司，可以解除劳动合同。

#### 2.5.3 员工即时解除
公司有下列情形之一的，员工可以解除劳动合同：
1. 未按照劳动合同约定提供劳动保护或者劳动条件的。
2. 未及时足额支付劳动报酬的。
3. 未依法为员工缴纳社会保险费的。
4. 公司的规章制度违反法律、法规的规定，损害员工权益的。
5. 因《劳动合同法》第二十六条第一款规定的情形致使劳动合同无效的。
6. 法律、行政法规规定员工可以解除劳动合同的其他情形。
公司以暴力、威胁或者非法限制人身自由的手段强迫员工劳动的，或者公司违章指挥、强令冒险作业危及员工人身安全的，员工可以立即解除劳动合同，不需事先告知公司。

#### 2.5.4 公司即时解除
员工有下列情形之一的，公司可以解除劳动合同：
1. 在试用期间被证明不符合录用条件的。
2. 严重违反公司的规章制度的。
3. 严重失职，营私舞弊，给公司造成重大损害的。
4. 员工同时与其他用人单位建立劳动关系，对完成本公司的工作任务造成严重影响，或者经公司提出，拒不改正的。
5. 因《劳动合同法》第二十六条第一款第一项规定的情形致使劳动合同无效的。
6. 被依法追究刑事责任的。

#### 2.5.5 公司提前通知解除
有下列情形之一的，公司提前30日以书面形式通知员工本人或者额外支付员工1个月工资后，可以解除劳动合同：
1. 员工患病或者非因工负伤，在规定的医疗期满后不能从事原工作，也不能从事由公司另行安排的工作的。
2. 员工不能胜任工作，经过培训或者调整工作岗位，仍不能胜任工作的。
3. 劳动合同订立时所依据的客观情况发生重大变化，致使劳动合同无法履行，经公司与员工协商，未能就变更劳动合同内容达成协议的。

#### 2.5.6 经济性裁员
有下列情形之一，需要裁减人员20人以上或者裁减不足20人但占公司职工总数10%以上的，公司提前30日向工会或者全体职工说明情况，听取工会或者职工的意见后，裁减人员方案经向劳动行政部门报告，可以裁减人员：
1. 依照企业破产法规定进行重整的。
2. 生产经营发生严重困难的。
3. 企业转产、重大技术革新或者经营方式调整，经变更劳动合同后，仍需裁减人员的。
4. 其他因劳动合同订立时所依据的客观经济情况发生重大变化，致使劳动合同无法履行的。

裁减人员时，应当优先留用下列人员：
1. 与本公司订立较长期限的固定期限劳动合同的。
2. 与本公司订立无固定期限劳动合同的。
3. 家庭无其他就业人员，有需要扶养的老人或者未成年人的。

公司依照本条第一款规定裁减人员，在6个月内重新招用人员的，应当通知被裁减的人员，并在同等条件下优先招用被裁减的人员。

#### 2.5.7 不得解除情形
员工有下列情形之一的，公司不得依照《劳动合同法》第四十条、第四十一条的规定解除劳动合同：
1. 从事接触职业病危害作业的员工未进行离岗前职业健康检查，或者疑似职业病病人在诊断或者医学观察期间的。
2. 在本公司患职业病或者因工负伤并被确认丧失或者部分丧失劳动能力的。
3. 患病或者非因工负伤，在规定的医疗期内的。
4. 女职工在孕期、产期、哺乳期的。
5. 在本公司连续工作满15年，且距法定退休年龄不足5年的。
6. 法律、行政法规规定的其他情形。

### 2.6 劳动合同终止
有下列情形之一的，劳动合同终止：
1. 劳动合同期满的。
2. 员工开始依法享受基本养老保险待遇的。
3. 员工死亡，或者被人民法院宣告死亡或者宣告失踪的。
4. 公司被依法宣告破产的。
5. 公司被吊销营业执照、责令关闭、撤销或者公司决定提前解散的。
6. 法律、行政法规规定的其他情形。

### 2.7 离职管理
#### 2.7.1 离职类型
1. 主动离职：员工主动提出解除劳动合同。
2. 被动离职：公司提出解除劳动合同。
3. 劳动合同终止：劳动合同期满或其他法定情形导致劳动合同终止。

#### 2.7.2 离职程序
1. **离职申请**：
   - 员工主动离职的，提前30日（试用期内提前3日）向公司提交书面离职申请。
   - 公司提出解除劳动合同的，向员工送达书面解除通知。
2. **离职审批**：
   - 部门负责人审核。
   - HR部门审核。
   - 按审批权限报相关领导审批。
3. **工作交接**：
   - 员工按照《工作交接清单》进行工作交接。
   - 交接内容包括工作内容、文件资料、物品设备、未完成事项等。
   - 交接双方和监交人签字确认。
4. **离职结算**：
   - 结算工资、奖金、加班费等。
   - 结算社会保险、住房公积金等。
   - 办理离职相关费用结算。
5. **离职手续办理**：
   - 交还公司财物。
   - 注销公司账号。
   - 签订保密协议、竞业限制协议（如适用）。
   - 领取离职证明。
6. **档案转移**：
   - 办理员工档案转移手续。
   - 办理社会保险、住房公积金转移手续。

#### 2.7.3 离职面谈
1. **面谈目的**：
   - 了解员工离职原因。
   - 听取员工对公司的意见和建议。
   - 解答员工的疑问。
   - 改善公司管理，减少人才流失。
2. **面谈人员**：
   - 部门负责人。
   - HR部门负责人。
   - 必要时公司高管。
3. **面谈内容**：
   - 离职原因。
   - 对公司的评价。
   - 对工作的评价。
   - 对管理的评价。
   - 意见和建议。

## 第3章 员工关系管理

### 3.1 员工沟通
1. **沟通渠道**：
   - 定期会议：部门例会、公司例会、全员大会等。
   - 一对一面谈：上级与下级的定期面谈。
   - 员工意见箱：收集员工的意见和建议。
   - 内部刊物：公司内部刊物，传达公司信息，分享员工故事。
   - 内部网络：公司内部网络平台，提供信息交流和沟通渠道。
   - 员工满意度调查：定期进行员工满意度调查，了解员工需求和意见。
2. **沟通要求**：
   - 及时沟通：及时传达公司信息，及时回应员工关切。
   - 有效沟通：确保沟通信息准确、完整、清晰。
   - 双向沟通：鼓励员工表达意见和建议，形成双向沟通机制。

### 3.2 员工关怀
1. **关怀内容**：
   - 生日关怀：员工生日时送上生日祝福和礼物。
   - 节日关怀：重要节日时送上节日祝福和慰问品。
   - 生病关怀：员工生病时进行探望和慰问。
   - 婚育关怀：员工结婚、生育时送上祝福和贺礼。
   - 困难关怀：员工遇到困难时提供帮助和支持。
2. **关怀方式**：
   - 物质关怀：提供物质上的帮助和支持。
   - 精神关怀：提供精神上的安慰和鼓励。
   - 活动关怀：组织各类活动，增强员工归属感。

### 3.3 员工活动
1. **活动类型**：
   - 文体活动：运动会、篮球赛、足球赛、羽毛球赛、歌唱比赛、舞蹈比赛等。
   - 团队活动：团建活动、户外拓展、旅游等。
   - 节日活动：春节晚会、中秋晚会、新年联欢会等。
   - 兴趣小组：读书小组、摄影小组、运动小组等。
2. **活动组织**：
   - 公司统一组织。
   - 部门组织。
   - 员工自发组织，公司提供支持。

### 3.4 员工权益保障
1. **劳动权益保障**：
   - 保障员工的劳动报酬权。
   - 保障员工的休息休假权。
   - 保障员工的劳动安全卫生权。
   - 保障员工的社会保险权。
   - 保障员工的职业培训权。
2. **民主权益保障**：
   - 保障员工的知情权。
   - 保障员工的参与权。
   - 保障员工的表达权。
   - 保障员工的监督权。

### 3.5 劳动争议处理
1. **处理原则**：
   - 合法原则。
   - 公正原则。
   - 及时原则。
   - 着重调解原则。
2. **处理方式**：
   - 协商：当事人双方自行协商解决。
   - 调解：向企业劳动争议调解委员会申请调解。
   - 仲裁：向劳动争议仲裁委员会申请仲裁。
   - 诉讼：向人民法院提起诉讼。
3. **处理程序**：
   - 发生劳动争议后，当事人可以协商解决。
   - 不愿协商、协商不成或者达成和解协议后不履行的，可以向调解组织申请调解。
   - 不愿调解、调解不成或者达成调解协议后不履行的，可以向劳动争议仲裁委员会申请仲裁。
   - 对仲裁裁决不服的，除法律另有规定的外，可以向人民法院提起诉讼。

## 第4章 附则

### 4.1 办法解释权
本办法由人力资源管理中心负责解释。

### 4.2 办法修订
本办法根据公司发展需要，适时进行修订。

### 4.3 办法生效
本办法自2026年1月1日起生效，原《员工关系与劳动合同管理办法》同时废止。
