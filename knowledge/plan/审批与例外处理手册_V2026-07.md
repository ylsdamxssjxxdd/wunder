# 审批与例外处理手册（V2026-07）

文档编号：`HR-EXCEPTION-HANDBOOK-2026-07`  
生效日期：`2026-01-01`  

# 第1章 总则

### 1.1 目的
本手册旨在规范公司各类审批与例外处理流程，建立健全审批机制，确保审批工作的规范性、高效性和公正性，保障公司各项业务的顺利开展。

### 1.2 适用范围
本手册适用于公司各类人力资源场景下的例外申请、审批、复核与归档，以及其他需要例外审批的业务场景。

### 1.3 基本原则
1. **合规性原则**：所有例外申请必须符合国家法律法规和公司规章制度。
2. **必要性原则**：例外申请必须确有必要，不得随意申请例外。
3. **审慎性原则**：例外审批必须审慎，严格审查申请理由和证据。
4. **透明性原则**：例外处理流程必须公开透明，保障员工知情权。
5. **可追溯性原则**：所有例外处理必须有完整的记录，可追溯、可审计。
6. **及时性原则**：例外申请必须及时处理，提高审批效率。

# 第2章 例外申请

### 2.1 例外申请的类型
例外申请分为以下类型：
1. **人力资源类例外**：
   - 任职时长豁免
   - 培训学时豁免
   - 认证有效期豁免
   - 绩效要求豁免
   - 任职资格豁免
   - 晋升破格申请
   - 薪酬调整例外
   - 招聘例外
   - 试用期豁免
   - 其他人力资源类例外
2. **业务类例外**：
   - 合同审批例外
   - 采购审批例外
   - 预算审批例外
   - 项目审批例外
   - 其他业务类例外
3. **行政类例外**：
   - 请假例外
   - 出差例外
   - 费用报销例外
   - 其他行政类例外

### 2.2 例外申请的条件
例外申请必须同时满足以下条件：
1. **存在特殊情况**：确有特殊情况，无法按常规制度执行。
2. **不违反法律法规**：例外申请不得违反国家法律法规。
3. **不损害公司利益**：例外申请不得损害公司利益。
4. **有充分的理由和证据**：例外申请必须有充分的理由和相关证据。
5. **符合公司战略和文化**：例外申请必须符合公司战略和文化价值观。

### 2.3 例外申请的提出
#### 2.3.1 申请主体
例外申请由以下主体提出：
1. 员工本人：涉及员工个人事项的例外申请，由员工本人提出。
2. 用人部门：涉及部门事项的例外申请，由用人部门负责人提出。
3. HRBP：协助员工或用人部门提出例外申请。
4. 其他相关部门：涉及其他业务事项的例外申请，由相关部门提出。

#### 2.3.2 申请时间
例外申请应在需要执行例外事项前提出，紧急情况可事后补申请，但需在3个工作日内完成。

#### 2.3.3 申请材料
例外申请应提交以下材料：
1. **例外申请表**：填写完整的例外申请表，包括：
   - 申请人信息
   - 申请日期
   - 例外类型
   - 涉及事项
   - 申请理由
   - 预期效果
   - 风险评估
   - 申请人签字
2. **相关证据材料**：
   - 证明特殊情况的文件
   - 相关数据和资料
   - 其他支持性材料
3. **补充说明材料**（如需要）：
   - 相关背景说明
   - 备选方案说明
   - 其他需要说明的事项

### 2.4 例外申请的填写要求
例外申请表的填写应遵循以下要求：
1. **真实性**：申请表内容必须真实、准确，不得弄虚作假。
2. **完整性**：申请表必填项必须全部填写，不得遗漏。
3. **清晰性**：申请表内容应表述清晰、逻辑清楚，易于理解。
4. **简洁性**：申请表内容应简洁明了，突出重点，避免冗长。
5. **规范性**：申请表填写应符合规范要求，字迹清晰（纸质版）或格式规范（电子版）。

### 3.2 审批流程
例外审批流程如下：

#### 3.2.1 一般审批流程
1. **申请提出**：申请人填写例外申请表，准备相关材料，提交申请。
2. **部门初审**：部门负责人对申请进行初审，审查申请的真实性、必要性和合理性。
   - 初审通过：签署意见后提交HR部门复审。
   - 初审不通过：退回申请人，说明理由。
3. **HR复审**：HR部门对申请进行复审，审查申请的合规性、政策依据和审批权限。
   - 复审通过：签署意见后提交相关领导审批。
   - 复审不通过：退回申请人或部门，说明理由。
4. **领导审批**：相关领导根据审批权限进行审批。
   - 审批通过：签署批准意见。
   - 审批不通过：退回并说明理由。
5. **结果通知**：HR部门将审批结果通知申请人和相关部门。
6. **执行与归档**：
   - 审批通过：申请人执行例外事项，HR部门将申请材料归档。
   - 审批不通过：申请人按常规制度执行。

#### 3.2.2 紧急审批流程
对于紧急情况，可启动紧急审批流程：
1. **申请提出**：申请人先通过口头或电话方式向相关领导申请。
2. **口头审批**：相关领导口头同意后，可先行执行。
3. **补填申请**：申请人在3个工作日内补填书面申请。
4. **补全流程**：按一般审批流程补全审批手续。
5. **归档**：HR部门将申请材料归档。

#### 3.2.3 特殊审批流程
对于特别重大、复杂的例外申请，启动特殊审批流程：
1. **申请提出**：申请人填写例外申请表，准备详细材料。
2. **专家评估**：组织相关专家对申请进行评估，形成评估意见。
3. **集体审议**：人力资源管理委员会或相关领导集体审议。
4. **审批决策**：根据集体审议结果做出审批决策。
5. **结果通知**：将审批结果通知申请人。
6. **归档**：HR部门将申请材料和评估意见归档。

### 3.3 审批时限
审批时限规定如下：
1. **部门初审**：2个工作日内完成。
2. **HR复审**：3个工作日内完成。
3. **领导审批**：5个工作日内完成。
4. **紧急审批**：口头审批立即响应，书面审批1个工作日内完成。
5. **特殊审批**：根据实际情况确定，一般不超过10个工作日。

如遇特殊情况，审批时限可适当延长，但需提前告知申请人。

### 3.4 审批意见
审批意见应包括以下内容：
1. **审批结论**：明确批准或不批准。
2. **审批意见**：说明审批的理由和依据。
3. **附加条件**（如批准）：说明附加条件和要求。
4. **后续要求**（如批准）：说明后续的跟踪和评估要求。
5. **审批人签字**：审批人签字并注明日期。

# 第4章 例外复议

### 4.1 复议申请
申请人对审批结果有异议的，可申请复议：

#### 4.1.1 复议条件
复议申请需满足以下条件：
1. 有新的证据或理由。
2. 认为原审批程序不合法或不公正。
3. 认为原审批依据不充分或不恰当。

#### 4.1.2 复议时限
复议申请应在收到审批结果后5个工作日内提出。

#### 4.1.3 复议材料
复议申请应提交以下材料：
1. **复议申请表**：填写复议申请表，包括：
   - 申请人信息
   - 申请日期
   - 原审批事项
   - 原审批结果
   - 复议理由
   - 新的证据和材料
   - 申请人签字
2. **原审批材料**：原例外申请表和审批意见。
3. **新的证据材料**：支持复议申请的新证据和材料。

### 4.2 复议流程
复议流程如下：
1. **复议申请**：申请人提交复议申请材料。
2. **复议受理**：HR部门收到复议申请后，2个工作日内决定是否受理。
   - 受理：通知申请人。
   - 不受理：说明理由。
3. **复议审查**：HR部门组织复议审查，审查内容包括：
   - 原审批程序是否合法合规。
   - 原审批依据是否充分恰当。
   - 新的证据和理由是否成立。
4. **复议决定**：根据审查结果做出复议决定：
   - 维持原审批决定：原审批决定正确，维持不变。
   - 变更原审批决定：原审批决定部分不当，进行变更。
   - 撤销原审批决定：原审批决定错误，予以撤销，重新审批。
5. **复议通知**：将复议决定通知申请人。
6. **执行与归档**：
   - 复议决定为最终决定，申请人必须执行。
   - HR部门将复议材料归档。

### 4.3 复议时限
复议时限规定如下：
1. **复议受理**：2个工作日内完成。
2. **复议审查**：5个工作日内完成。
3. **复议决定**：2个工作日内完成。
4. **复议通知**：1个工作日内完成。

复议总时限一般不超过10个工作日。

# 第5章 例外生效与执行

### 5.1 生效条件
例外条款需满足以下条件才能生效：
1. **审批通过**：例外申请必须按规定流程审批通过。
2. **在有效期内**：例外申请必须在有效期内执行。
3. **满足附加条件**（如有）：必须满足审批时附加的条件。
4. **不违反法律法规**：例外执行不得违反国家法律法规。
5. **不损害公司利益**：例外执行不得损害公司利益。

否则不生效。

### 5.2 有效期
例外申请的有效期规定如下：
1. **单次例外**：有效期为3个月，自审批通过之日起计算。
2. **短期例外**：有效期为6个月，自审批通过之日起计算。
3. **中期例外**：有效期为1年，自审批通过之日起计算。
4. **长期例外**：有效期为2年，自审批通过之日起计算，到期需重新申请。

具体有效期在审批时确定。

### 5.3 执行要求
例外执行应遵循以下要求：
1. **严格按审批内容执行**：不得超出审批范围执行例外事项。
2. **及时执行**：在有效期内及时执行例外事项。
3. **保留相关记录**：保留例外执行的相关记录和证据。
4. **定期报告**（长期例外）：定期向HR部门报告例外执行情况。
5. **接受监督检查**：接受HR部门和相关部门的监督检查。

### 5.4 执行监督
HR部门和相关部门对例外执行情况进行监督检查：
1. **定期检查**：定期检查例外执行情况。
2. **专项检查**：对重要例外事项进行专项检查。
3. **随机抽查**：随机抽查例外执行情况。
4. **员工举报**：接受员工对例外执行情况的举报。

# 第6章 例外归档与审计

### 6.1 归档范围
例外归档范围包括：
1. **例外申请表**：所有例外申请表。
2. **审批材料**：审批意见和相关文件。
3. **复议材料**：复议申请表和复议决定。
4. **执行材料**：例外执行的相关记录和证据。
5. **其他材料**：与例外相关的其他材料。

### 6.2 归档要求
例外归档应遵循以下要求：
1. **及时归档**：例外审批完成后10个工作日内归档。
2. **完整归档**：所有相关材料必须完整归档，不得遗漏。
3. **规范归档**：按规定的格式和要求归档。
4. **分类归档**：按例外类型和时间分类归档。
5. **电子归档**：建立电子档案，便于检索和管理。

### 6.3 保管期限
例外档案保管期限规定如下：
1. **一般例外**：保管期限为3年。
2. **重要例外**：保管期限为5年。
3. **重大例外**：保管期限为10年。
4. **涉及员工权益的例外**：保管期限为员工离职后2年。

### 6.4 审计要求
建立例外审计机制，定期对例外审批和执行情况进行审计：
1. **审计内容**：
   - 例外申请的真实性和必要性。
   - 例外审批的合规性和公正性。
   - 例外执行的规范性和有效性。
   - 例外归档的完整性和规范性。
2. **审计频率**：
   - 定期审计：每年审计一次。
   - 专项审计：根据需要进行专项审计。
3. **审计机构**：
   - 内部审计：由内部审计部门进行审计。
   - 外部审计：必要时聘请外部审计机构进行审计。
4. **审计报告**：
   - 审计完成后形成审计报告。
   - 审计报告报送公司管理层和相关部门。
5. **问题整改**：
   - 对审计发现的问题进行整改。
   - 跟踪整改情况，确保问题得到解决。

# 第7章 典型例外类型

### 7.1 任职时长豁免
#### 7.1.1 适用情形
任职时长豁免适用于以下情形：
1. 员工在公司外有丰富的相关工作经验，可抵扣任职时长。
2. 员工在公司内部有突出贡献，可破格缩短任职时长要求。
3. 员工因公司业务需要跨部门调动，任职时长可合并计算。
4. 其他经认定的特殊情形。

#### 7.1.2 申请要求
任职时长豁免申请需满足以下要求：
1. 提供相关工作经验证明。
2. 提供突出贡献的相关证据。
3. 说明豁免时长和理由。
4. 部门负责人和HRBP签字确认。

#### 7.1.3 审批权限
见第3.1.1条。

### 7.2 培训学时豁免
#### 7.2.1 适用情形
培训学时豁免适用于以下情形：
1. 员工在入职前已完成相关培训，可抵扣培训学时。
2. 员工通过自学或其他方式已掌握相关知识和技能，可豁免培训学时。
3. 员工因工作繁忙无法参加培训，可申请豁免并在后续补训。
4. 其他经认定的特殊情形。

#### 7.2.2 申请要求
培训学时豁免申请需满足以下要求：
1. 提供相关培训证明或自学证明。
2. 提供相关知识和技能掌握的证明。
3. 说明豁免学时和理由。
4. 如申请后续补训，需说明补训计划。

#### 7.2.3 审批权限
见第3.1.1条。

### 7.3 认证有效期豁免
#### 7.3.1 适用情形
认证有效期豁免适用于以下情形：
1. 认证考试因故延期，导致认证过期。
2. 员工因工作繁忙无法及时参加认证续期考试。
3. 认证机构更新认证标准，员工正在准备新认证考试。
4. 其他经认定的特殊情形。

#### 7.3.2 申请要求
认证有效期豁免申请需满足以下要求：
1. 提供认证过期的证明。
2. 说明豁免理由和续期计划。
3. 部门负责人签字确认。

#### 7.3.3 审批权限
见第3.1.1条。

### 7.4 项目紧急补位豁免
#### 7.4.1 适用情形
项目紧急补位豁免适用于以下情形：
1. 项目关键岗位人员突然离职或生病，需要紧急补位。
2. 项目时间紧急，需要破格使用不符合常规要求的人员。
3. 项目特殊，需要特殊技能的人员，而符合常规要求的人员不足。
4. 其他经认定的特殊情形。

#### 7.4.2 申请要求
项目紧急补位豁免申请需满足以下要求：
1. 说明项目紧急情况。
2. 说明补位人员情况和不符合常规要求的方面。
3. 说明补位的必要性和风险控制措施。
4. 项目负责人和部门负责人签字确认。

#### 7.4.3 审批权限
根据项目金额和影响程度确定，见第3.1.2条。

### 7.5 其他例外类型
其他例外类型包括但不限于：
- 绩效要求豁免
- 任职资格豁免
- 晋升破格申请
- 薪酬调整例外
- 招聘例外
- 试用期豁免
- 合同审批例外
- 采购审批例外
- 预算审批例外
- 项目审批例外
- 请假例外
- 出差例外
- 费用报销例外

具体要求和审批权限参照类似例外类型执行。

# 第8章 与中级晋升模拟关系

中级晋升模拟仅认可以下例外类型：
1. `PROM-MID-EXC-001`：任职时长豁免
2. `PROM-MID-EXC-002`：培训学时豁免
3. `PROM-MID-EXC-003`：必需认证豁免

其余例外类型不自动纳入该场景。

### 8.1 PROM-MID-EXC-001 任职时长豁免
#### 8.1.1 适用条件
1. 员工当前职级为初级，拟晋升中级。
2. 员工初级任职时长不足24个月，但满足以下条件之一：
   - 在公司外有3年以上相关工作经验。
   - 在公司内部有突出贡献，获得公司级奖励。
   - 因公司业务需要跨部门调动，原部门任职时长可合并计算。

#### 8.1.2 申请材料
1. 例外申请表。
2. 相关工作经验证明或突出贡献证明。
3. 部门负责人推荐意见。

#### 8.1.3 审批流程
1. 员工申请 → 部门负责人初审 → HR复审 → 人力资源总监审批。
2. 审批通过后，任职时长要求豁免为12个月。

### 8.2 PROM-MID-EXC-002 培训学时豁免
#### 8.2.1 适用条件
1. 员工当前职级为初级，拟晋升中级。
2. 员工2025年培训学时不足40学时，但满足以下条件之一：
   - 入职前已完成相关培训，培训学时可抵扣。
   - 通过自学或其他方式已掌握相关知识和技能，经考核合格。
   - 因工作繁忙无法参加培训，但已制定补训计划并承诺在晋升后6个月内完成。

#### 8.2.2 申请材料
1. 例外申请表。
2. 相关培训证明或考核合格证明。
3. 补训计划（如适用）。

#### 8.2.3 审批流程
1. 员工申请 → 部门负责人初审 → HR复审 → 人力资源总监审批。
2. 审批通过后，培训学时要求豁免为20学时。

### 8.3 PROM-MID-EXC-003 必需认证豁免
#### 8.3.1 适用条件
1. 员工当前职级为初级，拟晋升中级。
2. 员工未取得有效必需认证，但满足以下条件之一：
   - 认证考试已通过，证书正在办理中。
   - 已报名参加认证考试，考试时间在晋升后3个月内。
   - 因认证机构原因，认证考试延期或证书更新延迟。

#### 8.3.2 申请材料
1. 例外申请表。
2. 考试通过证明或考试报名证明。
3. 认证机构相关通知（如适用）。

#### 8.3.3 审批流程
1. 员工申请 → 部门负责人初审 → HR复审 → 人力资源总监审批。
2. 审批通过后，必需认证要求豁免，但需在晋升后6个月内取得认证。
