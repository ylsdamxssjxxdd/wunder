# 人才招聘与配置管理办法（V2026-07）

文档编号：`HR-RECRUITMENT-2026-07`  
生效日期：`2026-01-01`  

## 第1章 总则

### 1.1 目的
本办法旨在规范公司人才招聘与配置管理，建立科学、规范、高效的人才招聘与配置体系，吸引、选拔和配置优秀人才，优化人力资源配置，提高人力资源使用效率，为公司战略目标实现提供人才保障。

### 1.2 适用范围
本办法适用于公司全体正式员工的招聘与配置管理。劳务派遣员工、外包人员的招聘与配置参照执行。

### 1.3 基本原则
1. **战略导向原则**：招聘与配置必须服务于公司战略目标，支撑公司业务发展。
2. **公平公正原则**：招聘与配置必须公平公正，确保机会均等、程序透明。
3. **择优录用原则**：招聘必须择优录用，选拔最适合岗位的人才。
4. **人岗匹配原则**：配置必须实现人岗匹配，充分发挥员工的能力和潜能。
5. **效率优先原则**：招聘与配置必须高效，降低招聘成本，提高招聘效率。
6. **依法合规原则**：招聘与配置必须符合国家法律法规和公司规章制度。

## 第2章 招聘需求管理

### 2.1 招聘需求提出
招聘需求由用人部门提出，填写《招聘需求申请表》，说明以下内容：
1. 招聘岗位名称、岗位编码、所属部门。
2. 招聘人数、到岗时间。
3. 岗位职责、任职资格要求。
4. 招聘原因、岗位编制情况。
5. 薪酬预算范围。
6. 其他需要说明的事项。

### 2.2 招聘需求审批
招聘需求审批权限如下：
1. **编制内招聘**：
   - 普通员工：部门负责人审批，HR部门备案。
   - 主管级：业务分管领导审批，HR部门备案。
   - 经理级：CEO审批，HR部门备案。
   - 总监级及以上：董事会审批，HR部门备案。
2. **编制外招聘**：
   - 所有编制外招聘：CEO审批，HR部门备案。
   - 重要岗位编制外招聘：董事会审批，HR部门备案。

### 2.3 招聘计划制定
HR部门根据审批通过的招聘需求，制定招聘计划，招聘计划包括：
1. 招聘岗位清单。
2. 招聘时间安排。
3. 招聘渠道选择。
4. 招聘预算。
5. 招聘团队组成。
6. 其他相关事项。

## 第3章 招聘渠道管理

### 3.1 招聘渠道类型
公司采用多种招聘渠道，包括：
1. **内部招聘渠道**：
   - 内部竞聘
   - 内部推荐
   - 岗位轮换
   - 内部晋升
2. **外部招聘渠道**：
   - 网络招聘平台（前程无忧、智联招聘、BOSS直聘、猎聘等）
   - 校园招聘
   - 人才市场
   - 猎头公司
   - 社交媒体招聘（LinkedIn、微信、微博等）
   - 行业协会推荐
   - 其他招聘渠道

### 3.2 招聘渠道选择原则
招聘渠道选择遵循以下原则：
1. **岗位匹配原则**：根据岗位特点选择合适的招聘渠道。
2. **成本效益原则**：选择成本效益最优的招聘渠道。
3. **效率优先原则**：选择能快速找到合适人才的招聘渠道。
4. **质量保证原则**：选择能保证招聘质量的招聘渠道。

### 3.3 招聘渠道管理要求
1. **网络招聘平台**：
   - 定期更新招聘信息。
   - 及时查看和回复简历。
   - 定期评估招聘渠道效果。
2. **校园招聘**：
   - 提前制定校园招聘计划。
   - 选择目标院校和专业。
   - 做好校园宣讲和面试组织。
3. **猎头公司**：
   - 选择资质好、信誉高的猎头公司。
   - 签订正式的合作协议。
   - 明确服务内容和费用标准。
4. **内部推荐**：
   - 建立内部推荐奖励机制。
   - 及时处理内部推荐简历。
   - 定期反馈推荐结果。

## 第4章 招聘流程管理

### 4.1 简历筛选
1. **简历收集**：通过各招聘渠道收集简历。
2. **简历筛选**：HR部门根据任职资格要求对简历进行初步筛选。
3. **简历推荐**：HR部门将筛选通过的简历推荐给用人部门。
4. **简历审核**：用人部门对推荐的简历进行审核，确定面试人选。

### 4.2 面试组织
1. **面试通知**：HR部门向面试人选发送面试通知，包括面试时间、地点、联系方式、需准备的材料等。
2. **面试准备**：
   - 成立面试小组。
   - 准备面试材料（简历、评分表、问题清单等）。
   - 布置面试场地。
3. **面试实施**：
   - 签到登记。
   - 初步面试（HR面试）。
   - 专业面试（用人部门面试）。
   - 综合面试（高管面试，必要时）。
   - 背景调查（必要时）。

### 4.3 面试评估
1. **面试评分**：面试小组成员根据面试情况进行评分。
2. **面试讨论**：面试小组对面试人选进行讨论，形成评估意见。
3. **录用建议**：面试小组提出录用建议，包括录用人员、薪酬建议等。

### 4.4 录用审批
1. **录用申请**：HR部门填写《录用审批表》，附相关材料。
2. **录用审批**：按审批权限报相关领导审批。
3. **录用通知**：审批通过后，HR部门向拟录用人员发送录用通知。

### 4.5 入职管理
1. **入职准备**：
   - 通知拟录用人员准备入职材料。
   - 安排入职体检（必要时）。
   - 准备办公设备和工位。
2. **入职手续办理**：
   - 验证入职材料。
   - 签订劳动合同。
   - 办理入职登记。
   - 建立员工档案。
3. **入职培训**：
   - 公司级入职培训。
   - 部门级入职培训。
   - 岗位级入职培训。
4. **试用期管理**：
   - 制定试用期目标和计划。
   - 试用期辅导和跟踪。
   - 试用期考核评估。
   - 试用期转正或解除。

## 第5章 人才配置管理

### 5.1 人才配置原则
人才配置遵循以下原则：
1. **人岗匹配原则**：根据员工能力和岗位要求进行配置。
2. **用人所长原则**：充分发挥员工的优势和特长。
3. **优化组合原则**：实现团队成员的优化组合。
4. **动态调整原则**：根据业务变化和员工发展动态调整。

### 5.2 人才配置方式
人才配置方式包括：
1. **新员工配置**：根据新员工的能力和岗位要求进行配置。
2. **岗位轮换**：有计划地安排员工在不同岗位间轮换。
3. **内部调动**：根据工作需要，将员工从一个部门调动到另一个部门。
4. **晋升配置**：根据员工能力和绩效，将员工晋升到更高岗位。
5. **项目配置**：根据项目需要，从各部门抽调人员组成项目团队。

### 5.3 人才配置流程
1. **配置需求提出**：用人部门提出人才配置需求。
2. **配置方案制定**：HR部门制定人才配置方案。
3. **配置方案沟通**：与相关部门和员工沟通配置方案。
4. **配置方案审批**：按审批权限报相关领导审批。
5. **配置方案实施**：组织实施人才配置方案。
6. **配置效果跟踪**：跟踪人才配置效果，及时调整优化。

## 第6章 招聘效果评估

### 6.1 招聘效果评估指标
招聘效果评估指标包括：
1. **招聘效率指标**：
   - 招聘周期（从需求提出到人员到岗的时间）。
   - 招聘完成率（实际招聘人数/计划招聘人数）。
   - 简历有效率（有效简历数/收到简历总数）。
   - 面试通过率（通过面试人数/参加面试人数）。
2. **招聘质量指标**：
   - 试用期通过率（通过试用期人数/录用人数）。
   - 新员工绩效（新员工入职后第一个绩效周期的绩效）。
   - 新员工留存率（新员工入职一定时间后的留存比例）。
   - 用人部门满意度（用人部门对招聘工作的满意度）。
3. **招聘成本指标**：
   - 人均招聘成本（招聘总成本/录用人数）。
   - 招聘渠道成本（各招聘渠道的成本）。
   - 招聘预算执行率（实际招聘成本/预算招聘成本）。

### 6.2 招聘效果评估流程
1. **数据收集**：收集招聘相关数据。
2. **数据分析**：对收集的数据进行分析。
3. **评估报告**：形成招聘效果评估报告。
4. **反馈改进**：反馈评估结果，提出改进措施。

## 第7章 招聘纪律与监督

### 7.1 招聘纪律
1. 招聘人员必须严格遵守招聘纪律，不得泄露招聘机密。
2. 招聘人员必须公平公正地对待每一位应聘者，不得歧视任何应聘者。
3. 招聘人员不得利用职务之便谋取私利，不得接受应聘者的贿赂或好处。
4. 招聘人员必须严格遵守回避制度，与应聘者有亲属关系或利害关系的，应当主动回避。
5. 招聘人员必须严格按照招聘流程和标准进行招聘，不得随意简化流程或降低标准。

### 7.2 招聘监督
1. HR部门负责招聘工作的监督检查。
2. 公司纪检监察部门负责招聘工作的纪律监督。
3. 任何部门和个人都有权对招聘工作中的违规违纪行为进行举报。
4. 对举报的违规违纪行为，应当及时调查处理。

## 第8章 附则

### 8.1 办法解释权
本办法由人力资源管理中心负责解释。

### 8.2 办法修订
本办法根据公司发展需要，适时进行修订。

### 8.3 办法生效
本办法自2026年1月1日起生效，原《人才招聘与配置管理办法》同时废止。
