# 纪律与奖惩管理实施细则（V2026-07）

文档编号：`HR-DISCIPLINE-REWARD-2026-07`  
生效日期：`2026-01-01`  

## 第1章 总则

### 1.1 目的
本细则旨在规范公司员工行为，维护公司正常的生产经营秩序，保障公司和员工的合法权益，营造积极向上、遵纪守法的工作氛围，促进公司持续健康发展。

### 1.2 适用范围
本细则适用于公司全体正式员工。劳务派遣员工、外包人员、实习人员参照执行。

### 1.3 基本原则
1. **依法依规原则**：纪律与奖惩必须符合国家法律法规和公司规章制度。
2. **公平公正原则**：纪律与奖惩必须公平公正，对所有员工一视同仁。
3. **教育与惩戒相结合原则**：纪律处分必须坚持教育为主、惩戒为辅，帮助员工认识错误、改正错误。
4. **奖励与惩戒相结合原则**：既要奖励先进，也要惩戒违规，树立正确的导向。
5. **及时高效原则**：纪律与奖惩必须及时进行，提高时效性和威慑力。
6. **程序正当原则**：纪律与奖惩必须严格按照规定程序进行，保障员工的知情权、陈述权和申辩权。

## 第2章 员工行为规范

### 2.1 基本行为规范
1. **遵纪守法**：遵守国家法律法规，遵守公司规章制度，依法履行职责。
2. **爱岗敬业**：热爱本职工作，认真履行岗位职责，努力完成工作任务。
3. **诚实守信**：诚实守信，言行一致，不弄虚作假，不欺骗他人。
4. **团结协作**：尊重他人，团结同事，相互配合，共同完成工作任务。
5. **廉洁自律**：廉洁自律，不利用职务之便谋取私利，不接受贿赂。
6. **保守秘密**：保守国家秘密和公司商业秘密，不泄露公司机密信息。
7. **爱护公物**：爱护公司财物，合理使用公司资源，不浪费、不损坏。
8. **文明礼貌**：文明礼貌，举止得体，语言规范，树立良好形象。

### 2.2 工作纪律
1. **考勤纪律**：
   - 按时上下班，不迟到、不早退、不旷工。
   - 严格执行请假制度，按规定程序请假。
   - 工作时间不得擅自离岗、串岗、睡岗。
2. **工作秩序**：
   - 工作时间保持安静，不得大声喧哗、嬉戏打闹。
   - 工作时间不得从事与工作无关的活动，如玩游戏、看视频、炒股等。
   - 保持工作环境整洁，物品摆放整齐。
3. **会议纪律**：
   - 按时参加会议，不迟到、不早退、不无故缺席。
   - 会议期间保持安静，手机调至静音或震动状态。
   - 认真听讲，积极发言，遵守会议秩序。
4. **办公纪律**：
   - 正确使用办公设备和办公用品，节约使用资源。
   - 不得在办公区域吸烟、饮酒。
   - 遵守办公区域管理规定，维护办公秩序。

### 2.3 职业道德
1. **职业操守**：
   - 遵守职业操守，维护职业声誉。
   - 不从事与公司利益相冲突的活动。
   - 不利用职务之便为自己或他人谋取不正当利益。
2. **客户服务**：
   - 树立客户至上的服务理念，热情对待客户。
   - 认真倾听客户需求，及时解决客户问题。
   - 不得怠慢、刁难客户，不得与客户发生争执。
3. **同事关系**：
   - 尊重同事，平等相待，不歧视、不排斥他人。
   - 不得在背后议论、诽谤同事。
   - 不得拉帮结派、搞小团体。
4. **社会责任**：
   - 积极履行社会责任，参与公益活动。
   - 遵守社会公德，维护社会秩序。
   - 保护环境，节约资源。

### 2.4 安全保密
1. **安全生产**：
   - 遵守安全生产规章制度，落实安全生产责任。
   - 正确使用安全设施和劳动防护用品。
   - 发现安全隐患及时报告，不得隐瞒。
   - 不得违规操作，确保安全生产。
2. **消防安全**：
   - 遵守消防安全规定，落实消防安全责任。
   - 掌握消防知识，会使用消防器材。
   - 不得堵塞消防通道，不得损坏消防设施。
   - 发现火灾及时报警，积极扑救。
3. **信息安全**：
   - 遵守信息安全规定，保护公司信息系统安全。
   - 不得使用他人账号，不得泄露密码。
   - 不得访问非法网站，不得下载非法软件。
   - 及时更新杀毒软件，防范病毒攻击。
4. **保密纪律**：
   - 遵守保密纪律，保守国家秘密和公司商业秘密。
   - 不得泄露公司技术秘密、经营秘密、客户信息等。
   - 不得在公共场所谈论公司机密事项。
   - 离职时不得带走公司机密资料。

## 第3章 奖励

### 3.1 奖励类型
公司设立以下奖励类型：
1. **荣誉奖励**：
   - 通报表扬
   - 授予荣誉称号
   - 颁发荣誉证书
   - 授予奖章
2. **物质奖励**：
   - 奖金
   - 奖品
   - 股票期权
   - 特殊津贴
3. **发展奖励**：
   - 晋升
   - 调薪
   - 优先培训
   - 优先晋升
   - 国内外考察学习

### 3.2 奖励条件
员工有下列情形之一的，公司可以给予奖励：
1. **工作业绩突出**：
   - 超额完成工作任务，业绩显著。
   - 工作质量高，成果突出。
   - 为公司创造显著经济效益或社会效益。
2. **创新创造**：
   - 技术创新，取得重要成果。
   - 管理创新，提高管理效率。
   - 工艺创新，降低成本，提高质量。
   - 提出合理化建议，被采纳并取得显著效果。
3. **保护公司利益**：
   - 保护公司财产，防止或挽回重大损失。
   - 维护公司声誉，做出突出贡献。
   - 抵制歪风邪气，维护公司秩序。
4. **团队协作**：
   - 团队协作表现突出，带动团队共同进步。
   - 帮助同事解决困难，获得一致好评。
5. **见义勇为**：
   - 见义勇为，保护他人生命财产安全。
   - 抢险救灾，表现突出。
6. **其他贡献**：
   - 为公司做出其他突出贡献。
   - 其他应当给予奖励的情形。

### 3.3 奖励等级
奖励分为以下等级：
1. **嘉奖**：通报表扬，一次性奖金500-2000元。
2. **记功**：通报表扬，颁发荣誉证书，一次性奖金2000-10000元。
3. **记大功**：通报表扬，颁发荣誉证书，一次性奖金10000-50000元。
4. **授予荣誉称号**：
   - 优秀员工
   - 优秀管理者
   - 技术标兵
   - 销售冠军
   - 劳动模范
   - 其他荣誉称号
5. **特殊贡献奖**：通报表扬，颁发荣誉证书和奖章，一次性奖金50000元以上，或给予股票期权等长期激励。

### 3.4 奖励程序
奖励程序如下：
1. **推荐**：员工本人、部门或其他相关方推荐。
2. **申报**：填写《奖励审批表》，并附相关证明材料。
3. **审核**：所在部门负责人审核，HR部门复核。
4. **评审**：成立奖励评审委员会进行评审。
5. **审批**：按审批权限报相关领导审批。
6. **公示**：对拟奖励人员进行公示，公示期3-7天。
7. **实施**：发布奖励决定，实施奖励。
8. **归档**：奖励材料存入员工档案。

### 3.5 奖励权限
奖励审批权限如下：
1. **嘉奖**：部门负责人审批，HR部门备案。
2. **记功**：业务分管领导审批，HR部门备案。
3. **记大功**：CEO审批，HR部门备案。
4. **授予荣誉称号**：CEO审批，HR部门备案。
5. **特殊贡献奖**：董事会审批，HR部门备案。

## 第4章 惩戒

### 4.1 惩戒类型
公司设立以下惩戒类型：
1. **口头警告**：对轻微违规行为进行口头批评教育。
2. **书面警告**：对违规行为进行书面警告，记入员工档案。
3. **记过**：对较重违规行为进行记过处分，记入员工档案。
4. **记大过**：对严重违规行为进行记大过处分，记入员工档案。
5. **降职/降级**：降低职务级别或工资级别。
6. **停职检查**：暂停职务，进行检查反省。
7. **解除劳动合同**：解除劳动关系，辞退员工。

### 4.2 惩戒措施
除上述惩戒类型外，还可以采取以下惩戒措施：
1. **经济处罚**：
   - 扣发绩效奖金
   - 降薪
   - 赔偿经济损失
2. **其他措施**：
   - 取消评优评先资格
   - 取消晋升资格
   - 取消培训资格
   - 调离原岗位
   - 其他必要措施

### 4.3 违规行为分类
违规行为分为以下几类：

#### 4.3.1 轻微违规行为
有下列情形之一的，属于轻微违规行为：
1. 迟到、早退10分钟以内，每月不超过2次。
2. 工作时间擅自离岗、串岗10分钟以内。
3. 工作时间从事与工作无关的活动，情节较轻。
4. 未按规定着装，情节较轻。
5. 办公区域物品摆放杂乱，经提醒后未及时整改。
6. 会议迟到、早退，情节较轻。
7. 其他情节轻微的违规行为。

#### 4.3.2 一般违规行为
有下列情形之一的，属于一般违规行为：
1. 迟到、早退10分钟以上30分钟以内，或每月迟到、早退3-5次。
2. 旷工半天以内。
3. 工作时间擅自离岗、串岗10分钟以上30分钟以内。
4. 工作时间从事与工作无关的活动，情节较重。
5. 未按规定履行请假手续，擅自离岗。
6. 不服从工作安排，情节较轻。
7. 工作责任心不强，出现工作失误，造成一定影响。
8. 违反会议纪律，情节较重。
9. 泄露公司一般信息，造成一定影响。
10. 其他情节一般的违规行为。

#### 4.3.3 较重违规行为
有下列情形之一的，属于较重违规行为：
1. 迟到、早退30分钟以上，或每月迟到、早退6次以上。
2. 旷工1-2天。
3. 工作时间擅自离岗、串岗30分钟以上。
4. 工作时间玩游戏、看视频、炒股等，情节严重。
5. 不服从工作安排，与上级发生争执，造成不良影响。
6. 工作责任心不强，出现严重工作失误，造成较大损失。
7. 违反保密规定，泄露公司重要信息，造成较大损失。
8. 违反安全生产规定，造成安全隐患或一般安全事故。
9. 弄虚作假，欺骗公司，情节较重。
10. 诽谤、诋毁他人，造成不良影响。
11. 其他情节较重的违规行为。

#### 4.3.4 严重违规行为
有下列情形之一的，属于严重违规行为：
1. 连续旷工3天以上，或年度累计旷工5天以上。
2. 严重不服从工作安排，拒不执行工作指令。
3. 工作严重失职，造成重大损失或恶劣影响。
4. 违反保密规定，泄露公司核心机密，造成重大损失。
5. 违反安全生产规定，造成重大安全事故。
6. 贪污、受贿、挪用公款，或利用职务之便谋取私利。
7. 盗窃、侵占公司财物。
8. 欺诈、弄虚作假，骗取公司利益。
9. 打架斗殴，造成人身伤害或财产损失。
10. 吸毒、赌博、嫖娼等违法行为。
11. 被依法追究刑事责任。
12. 严重违反职业道德，给公司造成重大损害。
13. 其他严重违反公司规章制度的行为。

### 4.4 惩戒标准
惩戒标准如下：

| 违规行为类型 | 首次违规 | 再次违规 | 多次违规 |
|-----------|---------|---------|---------|
| 轻微违规 | 口头警告 | 书面警告 | 记过 |
| 一般违规 | 书面警告 | 记过 | 记大过 |
| 较重违规 | 记过 | 记大过 | 降职/降级、停职检查 |
| 严重违规 | 解除劳动合同 | - | - |

### 4.5 惩戒程序
惩戒程序如下：
1. **发现违规**：发现员工违规行为，及时报告。
2. **调查核实**：成立调查组，对违规行为进行调查核实。
3. **听取陈述申辩**：听取员工的陈述和申辩。
4. **形成处理意见**：根据调查结果，形成处理意见。
5. **审批**：按审批权限报相关领导审批。
6. **告知**：将处理决定书面告知员工本人。
7. **实施**：实施处理决定。
8. **归档**：处理材料存入员工档案。

### 4.6 惩戒权限
惩戒审批权限如下：
1. **口头警告**：部门负责人审批，HR部门备案。
2. **书面警告**：部门负责人审批，HR部门备案。
3. **记过**：业务分管领导审批，HR部门备案。
4. **记大过**：CEO审批，HR部门备案。
5. **降职/降级、停职检查**：CEO审批，HR部门备案。
6. **解除劳动合同**：CEO审批，HR部门备案，必要时报董事会。

## 第5章 申诉与复核

### 5.1 申诉
员工对惩戒决定不服的，可以在收到处理决定之日起15日内，向HR部门提出书面申诉。申诉应当说明申诉理由和请求，并附相关证据材料。

### 5.2 复核
HR部门收到申诉后，应当在15日内组成复核小组进行复核。复核小组应当听取申诉人的陈述和申辩，查阅相关材料，进行必要的调查。

### 5.3 复核决定
复核小组应当在30日内作出复核决定：
1. 原处理决定认定事实清楚、证据确凿、适用依据正确、程序合法、处理适当的，维持原处理决定。
2. 原处理决定有下列情形之一的，撤销或变更原处理决定：
   - 认定事实不清、证据不足的。
   - 适用依据错误的。
   - 违反法定程序的。
   - 处理明显不当的。

### 5.4 执行
复核决定为最终决定，员工应当执行。复核期间，不停止原处理决定的执行。
