# 气象风险提示话术

## 元信息

- 分类：服务表达
- 关键词：风险提示、公众服务、防御建议、不确定性、短句模板、离线话术
- 来源等级：B
- 适用范围：离线生成稳妥的气象风险提示
- 使用边界：不能替代官方预警、应急命令或行业调度

## 通用话术结构

```markdown
受……影响，可能出现……天气。需关注……风险。建议……。当前判断仍需结合当地气象部门最新预报预警和实时观测核验。
```

## 分灾种表达

### 暴雨

关注短时强降水、持续降水和局地积水风险。建议减少涉水出行，避开低洼区域、地下空间、河道沟谷和易发生地质灾害区域。

### 雷暴大风

关注阵风、雷电和局地强对流风险。建议远离临时搭建物、广告牌、树木和高空悬挂物，户外活动需及时转移。

### 冰雹

关注车辆、棚架、农作物和户外人员风险。建议提前转移易损物品，减少露天停留。

### 高温

关注中暑、用电负荷和户外作业风险。建议避开高温时段，补水降温，关注脆弱人群。

### 大雾

关注低能见度和道路湿滑风险。建议降低车速，保持车距，关注交通管制信息。

## 禁用表达

- “一定会发生灾害”
- “可以解除警报”
- “我发布预警”
- “无需关注官方信息”
- “只要看这个指标就能判断”
