# 通用气象知识检索索引

## 使用方式

先用本索引判断主题，再用 `搜索内容` 定点检索。常用检索语法：

```text
搜索范围：{{SKILL_ROOT}}/references
关键词：法规名、天气类型、观测产品、业务动作、同义词
```

不要把本技能当作实时资料源。涉及现行规则、地方标准、灾害预警和法律责任时，必须核验官方原文。

## 输出时的依据标注

建议格式：

```markdown
依据：通用气象知识/references/01-法规政策/index.md；来源等级：S/A；现行性：需以官方原文核验。
```

如果只找到 B/C 级材料，不要写成确定法规结论。应写：

```markdown
当前本地资料只支持经验性判断，未检索到可直接支撑该结论的法规原文。
```
