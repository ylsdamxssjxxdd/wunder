# 自动站与资料质控

## 元信息

- 分类：观测资料
- 关键词：自动站、地面观测、雨量、风、温湿、气压、质控、代表性、缺测、极端值
- 来源等级：B
- 适用范围：离线解释自动站资料使用和质量控制
- 使用边界：不含具体站点元数据和实时观测

## 自动站资料的价值

自动站是核验天气影响的重要实况资料，尤其适合验证短时强降水、雷暴大风、降温、能见度和气压变化。但站点资料受代表性和质量控制影响，不能不加判断地外推到大范围。

## 质控思路

| 问题 | 检查方式 |
| --- | --- |
| 极端值是否可信 | 看相邻站、前后时次和天气背景是否一致 |
| 是否缺测或跳变 | 检查时间序列连续性 |
| 站点是否有代表性 | 关注地形、城市环境、海拔、周边遮挡 |
| 降水是否局地 | 与雷达回波和周边站雨量交叉验证 |
| 大风是否真实 | 检查阵风时次、风向突变和雷达/天气背景 |

## 常见使用误区

- 用单站代表整个区域。
- 把设备异常当成极端天气。
- 只看小时雨量，忽略分钟级短时雨强。
- 忽略站点海拔和下垫面差异。
