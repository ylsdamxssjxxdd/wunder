## 地面观测

自动气象站常见要素包括温度、湿度、风、气压、降水、能见度等。使用时要关注：

- 站点代表性：站点环境可能不能代表整个城市或山区。
- 时间分辨率：分钟级、小时级、日值的含义不同。
- 质控状态：极端值可能是真实天气，也可能是设备或传输异常。
- 空间差异：强对流和短时强降水具有很强局地性。

## 天气雷达基础

| 产品 | 常见用途 | 局限 |
| --- | --- | --- |
| 反射率 | 判断降水回波强弱、结构、移动和组织形态 | 不能直接等同地面雨强；受距离、波束高度、亮带和非气象回波影响 |
| 径向速度 | 判断相对雷达的风场、辐合、辐散、旋转、切变 | 只能观测沿雷达径向分量；存在速度模糊和距离折叠 |
| 谱宽 | 判断速度离散度、湍流、切变或资料质量问题 | 高谱宽可能来自真实湍流，也可能来自噪声或混合目标 |
| 双偏振产品 | 辅助识别水凝物类型、冰雹、融化层、非气象目标 | 产品含义依赖雷达体制、标定和业务算法 |

### 常见雷达误区

- 反射率颜色越强，不一定灾害越强；需要结合高度、速度、环境和地面实况。
- 远距离回波所在高度较高，不能直接代表近地面天气。
- 亮带可导致层状降水区反射率偏强，容易误判为强对流。
- 异常传播、地物、昆虫、鸟群等可能产生非气象回波。

## 卫星云图基础

| 类型 | 主要用途 | 注意点 |
| --- | --- | --- |
| 可见光 | 白天识别云系形态、纹理、低云和对流发展 | 夜间不可用或受限 |
| 红外 | 识别云顶温度、深对流云顶、台风云系 | 冷云顶不一定对应最强地面降水 |
| 水汽 | 分析中高层水汽输送、干侵入和环流结构 | 不能直接替代低层湿度观测 |
| 多通道合成 | 辅助识别云相、沙尘、雾、火点等 | 需结合业务产品说明和区域经验 |

## 闪电定位

闪电资料适合识别雷暴活跃程度和对流增强趋势。使用时注意：

- 闪电频次快速增加常提示对流发展。
- 云闪、地闪和定位误差需要区分。
- 无闪电不代表没有强降水、大风或积冰风险。

## 资料融合建议

解释天气过程时优先融合：

1. 大尺度背景：环流形势、锋面、低涡、台风、冷空气。
2. 中尺度触发：辐合线、地形、海陆风、边界层不稳定。
3. 实时观测：雷达、卫星、自动站、闪电。
4. 影响信息：已发生灾情、交通、设施和人群暴露度。
