# 暴雨与短时强降水

## 元信息

- 分类：灾害天气
- 关键词：暴雨、短时强降水、低空急流、水汽输送、训练效应、地形增雨、内涝、山洪
- 来源等级：B
- 适用范围：离线解释暴雨形成、识别思路和风险服务
- 使用边界：不含实时雨量、预警阈值和地方灾害风险等级

## 核心结论

暴雨通常需要水汽、抬升和持续机制共同作用。短时强降水强调短时间高雨强，累计暴雨强调长时间或多轮降水叠加。风险不只取决于雨量，还取决于落区、地形、排水能力和前期土壤含水量。

## 形成条件

| 条件 | 典型表现 | 判读意义 |
| --- | --- | --- |
| 水汽 | 暖湿输送、低空急流、水汽辐合 | 决定可降水量和持续补给 |
| 抬升 | 锋面、低涡、切变线、辐合线、地形 | 触发和维持降水 |
| 不稳定 | 对流能量、暖湿下垫面、冷暖空气叠置 | 决定短时雨强潜力 |
| 持续机制 | 训练效应、准静止系统、上游新生 | 决定累计雨量和灾害放大 |

## 重点风险

- 城市内涝：短时雨强超过排水能力，低洼地、下穿通道、地下空间风险高。
- 山洪和地质灾害：山区、沟谷、陡坡和前期降水偏多区域风险更高。
- 中小河流洪水：持续降水或上游来水叠加时风险增加。
- 交通影响：积水、能见度下降、道路冲毁或塌方。

## 服务表达要点

不要只说“有暴雨”。应说明：

- 降水是短时强、持续性强，还是多轮叠加。
- 风险是否集中在夜间、山区、低洼地或交通高暴露场景。
- 当前判断缺少哪些实时资料，例如雷达、自动站雨量、河流水位和地质灾害风险信息。
