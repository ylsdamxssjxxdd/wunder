# 寒潮、雨雪和冰冻

## 元信息

- 分类：灾害天气
- 关键词：寒潮、强降温、冷空气、雨雪、冻雨、道路结冰、低温冰冻
- 来源等级：B
- 适用范围：离线解释低温雨雪冰冻天气形成和风险
- 使用边界：不含实时温度、相态预报和官方预警阈值

## 核心结论

寒潮风险不只来自最低气温，还来自降温幅度、持续时间、大风、雨雪相态和地面结冰。雨雪冰冻的难点在于垂直温度结构和近地面温度差异，离线只能讲判读思路。

## 关键机制

- 冷空气南下：带来降温、大风和气压变化。
- 暖湿气流配合：提供降水条件，可能形成雨雪或冻雨。
- 垂直温度层结：决定雨、雪、雨夹雪、冻雨等相态。
- 地表温度：决定道路结冰、积雪和冻融影响。

## 风险类型

| 风险 | 关注点 |
| --- | --- |
| 强降温 | 脆弱人群、农业、能源和供暖需求 |
| 大风降温 | 体感温度下降、设施和水面风险 |
| 雨雪 | 交通延误、道路湿滑、积雪压损 |
| 冻雨 | 电线覆冰、树木倒伏、道路结冰 |
| 持续低温 | 作物冻害、供水设施冻损、能源负荷 |

## 服务表达

低温服务要把“降温幅度、最低气温、持续时间、风、雨雪相态、结冰风险”分开说。不要只写“有寒潮”，也不要在没有实时预报时给具体最低温度。
