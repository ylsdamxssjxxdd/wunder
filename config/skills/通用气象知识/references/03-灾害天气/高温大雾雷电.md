# 高温、大雾和雷电

## 元信息

- 分类：灾害天气
- 关键词：高温、热浪、湿球温度、城市热岛、大雾、低能见度、雷电、防雷
- 来源等级：B
- 适用范围：离线解释高温、大雾、雷电的形成和风险服务
- 使用边界：不含实时温度、能见度、雷电定位和防雷资质判断

## 高温与热浪

高温风险受最高气温、最低气温、湿度、持续天数和暴露场景共同影响。夜间最低气温偏高会削弱人体恢复，湿度高会降低散热效率。

服务重点：

- 户外作业和运动人群。
- 老年人、儿童、慢性病人等脆弱人群。
- 电力负荷、设备散热、道路和设施运行。
- 城市热岛和局地通风条件。

## 大雾与低能见度

大雾形成常需要水汽充足、近地层冷却、风力较小和层结稳定。平流雾、辐射雾、锋面雾和蒸发雾机制不同，消散条件也不同。

服务重点：

- 能见度变化是否突然。
- 低洼地、近水面、桥梁、隧道口和交通枢纽。
- 是否伴随道路湿滑、污染累积或低云。

## 雷电

雷电风险取决于雷暴发展和目标暴露度。防雷服务要强调避险行为，不夸大单点命中概率。

通用建议：

- 雷暴临近时减少开阔地、水面、高处和孤立树下停留。
- 户外活动需要留意雷声、闪电和官方预警。
- 防雷装置、检测资质和责任问题应回到法规政策专题核验。
