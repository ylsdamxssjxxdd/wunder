# 通用回波案例索引

## 使用方式

先按用户问题提取标签，再用标签检索案例。标签至少包含一个天气类型或风险类型，最好再加一个回波形态。

示例：

```text
问题：回波呈弓形，后侧有弱回波槽，可能是什么风险？
检索词：弓形回波 后侧入流 雷暴大风 下击暴流
```

## 组合检索建议

| 用户线索 | 推荐检索词 |
| --- | --- |
| 回波前缘呈弧形、风突然增强 | 弓形回波 后侧入流 雷暴大风 下击暴流 |
| 单体右后侧有钩状结构 | 钩状回波 中气旋 径向速度 旋转 |
| 同一地点降水持续很久 | 训练效应 准静止 短时强降水 暴雨 |
| 线状回波快速东移或南压 | 飑线 线状对流 阵风锋 速度辐合 |
| 反射率很强但是否有冰雹不确定 | 冰雹 强回波核 弱回波区 双偏振 |
| 雷达图上大片虚假回波 | 非气象回波 异常传播 地物 质控 |

## 案例使用边界

- 案例是训练和对照材料，不是实时诊断结论。
- 相同形态在不同季节、地形、雷达距离、扫描高度下含义可能不同。
- 业务判断必须结合环境场、连续体扫、地面实况和官方发布体系。
