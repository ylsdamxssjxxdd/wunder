# 回波判读框架

## 最小判读流程

1. 识别产品：反射率、径向速度、谱宽、双偏振、组合产品还是截图。
2. 确认时空：雷达站、扫描仰角、时次、目标距离、是否连续演变。
3. 看结构：单体、线状、片状、螺旋雨带、层状区、非气象回波。
4. 看演变：初生、增强、合并、成熟、减弱、后侧入流、训练效应。
5. 看耦合：反射率结构是否与速度辐合、旋转、地面风雨实况一致。
6. 做边界判断：哪些结论能说，哪些必须等待更多资料。

## 主要产品要点

| 产品 | 关注点 | 常见陷阱 |
| --- | --- | --- |
| 反射率 | 强回波核、弓形、钩状、线状、层状区、回波顶 | 高反射率不等于地面灾害一定发生 |
| 径向速度 | 辐合、辐散、旋转、阵风锋、后侧入流 | 只反映沿雷达径向速度，易受速度模糊影响 |
| 谱宽 | 切变、湍流、混合目标、资料质量 | 高谱宽可能是噪声或非气象目标 |
| 双偏振 | 水凝物类型、冰雹、融化层、非气象目标 | 需要结合本地算法、标定和业务经验 |
| 连续体扫 | 移速、移向、增强减弱、合并分裂 | 单张图像容易误判生命周期 |

## 相似性匹配维度

匹配案例时按四层比较：

- `形态相似`：回波外观、组织方式、强弱分布是否类似。
- `动力相似`：速度产品是否显示辐合、旋转、下沉外流或后侧入流。
- `环境相似`：水汽、能量、垂直风切变、触发机制是否接近。
- `影响相似`：地面风雨、冰雹、积水、灾情或用户关心的影响是否一致。

只有形态相似而动力和环境不相似时，输出必须降级为“外观相似，风险不能直接外推”。

## 常见误判

- 把亮带误判为强对流。
- 把异常传播或地物回波误判为降水。
- 把远距离高空强回波直接等同于近地面强降水。
- 把钩状形态直接等同于龙卷。
- 把弓形回波直接等同于必然大风。
- 忽略速度产品和地面实况，只看反射率颜色。
