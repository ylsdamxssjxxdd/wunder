# 自检清单

## 结构检查
- 标题是否明确反映文稿类型与研究对象。
- 研究问题、资料范围、方法说明是否出现在前部章节。
- 各章节是否围绕同一条主线推进，而不是东拼西凑。

## 证据检查
- 每个关键结论是否都能回到已知材料。
- 是否存在只讲观点、不讲资料基础的段落。
- 引用和参考文献是否一一对应。
- 是否有“待核对”信息被误写成确定事实。

## 写作检查
- 是否存在大量只有列表没有分析的段落。
- 章节间是否有过渡句说明逻辑关系。
- 术语是否前后一致。
- 是否滥用口语化、宣传化或绝对化表达。

## 修订检查
- 如果是答复评审意见，是否逐条回应。
- 是否明确写出修改动作和修改位置。
- 对未采纳意见是否给出可辩护理由。

## 导出检查
- 第一行是否为正式标题。
- 标题、表格、代码块前后是否留空行。
- 是否残留多余 Markdown 符号或临时批注。
