# 引用核对与参考文献整理

## 用途
这份文件用于把“正文里提到的来源”和“参考文献里列出的条目”真正对齐，避免成稿阶段最常见的引用错漏。

## 适用场景
- 正文已经写完，需要统一检查参考文献。
- 来源来自本地材料、截图、摘录、内部报告，信息不完整。
- 需要把草稿里的“待补充来源”收口成正式参考文献区。

## 核心原则
- 正文引用先于参考文献排版。
- 没在正文里用到的来源，不要无意义地塞进参考文献。
- 不能核实的信息，不要假装完整。
- 参考文献条目是为正文服务，不是装饰页数。

## 一次完整核对的顺序
1. 从正文里圈出所有显式引用、隐含引用和关键事实来源。
2. 给每一条正文引用建立唯一标识。
3. 检查参考文献区是否存在对应条目。
4. 检查条目中的作者、年份、标题、来源类型是否足以识别。
5. 对缺失项标记“待核对”，不要编造。

## 最低可接受条目
如果当前无法形成完整学术格式，至少保留：
- 作者或机构
- 标题或材料名
- 年份或时间范围
- 来源类型
- 可追溯线索，例如文件名、章节、链接、档案编号

## 建议字段
| 字段 | 说明 |
| --- | --- |
| 引用标识 | 正文中对应的引用编号或标签 |
| 作者/机构 | 谁产生了该来源 |
| 年份 | 来源时间 |
| 标题 | 文献、报告、材料或记录标题 |
| 来源类型 | 论文、报告、纪要、网页、数据、访谈记录等 |
| 载体信息 | 期刊、会议、机构、文件路径、URL 等 |
| 当前状态 | 已核实 / 待核对 / 缺失 |

## 常见错误
- 正文写了某研究，但参考文献里没有。
- 参考文献列了很多条目，正文完全没引用。
- 用“某研究指出”但没有任何可追溯信息。
- 把二手转述写成原始来源。
- DOI、页码、年份不确定却写成确定值。

## 离线场景建议
- 若引用来源是本地文件，优先写明文件名与章节。
- 若来源是内部材料，写明材料类型和时间。
- 若来源暂时只知道标题，保留标题并标注待核对，不要补假 DOI。
