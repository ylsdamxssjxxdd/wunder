# 证据分级与来源评估

## 用途
这份文件用于判断一条材料到底能不能支撑正文里的结论，以及应该支撑到什么强度。

## 一条基本原则
权威不等于证据，篇幅不等于可靠，观点鲜明也不等于可采纳。

## 七级证据结构
### 一级
系统综述、Meta 分析、严格汇总性研究。

### 二级
随机对照、强实验设计、严格控制的干预研究。

### 三级
无随机但有对照的准实验研究。

### 四级
队列、回顾、案例对照等观察性研究。

### 五级
对描述性或质性研究做系统整合的综述。

### 六级
单项描述性研究、个案研究、访谈研究、单次调查、单份案例分析。

### 七级
专家意见、委员会意见、立场性报告、社论、未经严格验证的行业判断。

## 对不同学科的解释
- 医学、健康类：更看重一到三级证据。
- 教育、社会科学：四到六级材料也常见，但要降低因果推断强度。
- 政策研究：正式文件和专家报告常重要，但不能直接替代实证证据。
- 人文研究：原始文本和解释框架本身就是核心证据，不必硬套实验逻辑。

## 单条来源的五项判断
### 1. 来源类型
- 正式论文
- 正式报告
- 政策文件
- 会议纪要
- 访谈记录
- 媒体报道
- 评论性文章

### 2. 方法是否写清楚
- 数据怎么来的
- 样本是什么
- 时间范围是什么
- 结论如何得出

### 3. 材料是否足以支持结论
- 能支持事实描述
- 能支持相关性判断
- 能支持机制解释
- 能支持因果判断

### 4. 时效性是否匹配
- 老材料在快变化领域可能只能做背景。
- 老材料在理论史、制度史问题上仍可能有效。

### 5. 利益或立场偏差
- 是否带有明显宣传目标。
- 是否与研究对象存在直接利益关系。
- 是否回避对自己不利的信息。

## 建议分级
### A 级
可作为主要证据。
条件：
- 来源清晰
- 方法清楚
- 结论与证据匹配
- 时效性合适

### B 级
可作为辅助证据。
条件：
- 基本可信
- 但证据强度或范围有限

### C 级
可用，但必须带保留条件。
条件：
- 材料能提供线索
- 但不足以单独支撑强结论

### D 级
仅在缺少更好材料时谨慎引用。
条件：
- 来源弱
- 方法不清
- 但问题本身又无法完全绕开

### F 级
不建议作为论证证据使用。
条件：
- 无法核实
- 明显失真
- 与结论强度严重不匹配

## 快速判断表
| 问题 | 是 | 否 |
| --- | --- | --- |
| 来源能识别吗 | 继续评估 | 降到 D/F |
| 方法说得清吗 | 继续评估 | 降一级 |
| 结论强度与证据匹配吗 | 可用 | 降一级 |
| 时效性合适吗 | 保留 | 仅作背景 |
| 是否存在明显立场偏差 | 加注说明 | 正常使用 |

## 在正文中的用法
- A 级：可直接作为核心论证支撑。
- B 级：适合补强、对照、说明趋势。
- C 级：必须加“现有材料显示”“初步表明”“仍待进一步核实”等限定语。
- D/F 级：最好不要进主结论，只能作为背景、争议点或批判对象。

## 常见错误
- 用单一案例推出普遍规律。
- 用描述性材料推出因果关系。
- 用利益相关方报告证明自身立场的普遍正确。
- 把二手转述当成一手证据。
- 看见正式格式就默认可靠，不审方法。
