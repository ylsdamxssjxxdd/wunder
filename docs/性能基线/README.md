# 性能基线目录

本目录保存性能优化、压测、回归防护相关的前后对比报告。详细标准见 [性能标准](../性能标准.md)。

## 命名

- 报告：`YYYY-MM-DD-<scope>-<topic>.md`
- 原始资料：`assets/YYYY-MM-DD-<scope>-<topic>-<kind>.<ext>`

`scope` 使用简短范围，如 `backend`、`frontend`、`docker`、`runtime`、`workspace`、`web`。
