# 管理端面板指南

管理端是系统管理员的工作台，用于配置系统、管理用户、监控运行状态。这页帮你快速定位各面板的功能。

## 系统分组

这个分组聚焦全局管理：

### 内部状态

查看系统运行状态，包括：
- 活跃会话数
- 系统负载
- 关键组件状态

### 系统设置

配置全局参数，包括：
- 服务地址和端口
- 安全设置
- 存储设置
- 工具开关

### 单位管理

管理组织架构：
- 创建和管理单位
- 单位层级关系
- 单位成员管理

### 用户管理

管理用户账户：
- 用户列表查看
- 创建新用户
- 编辑用户信息
- Token 账户管理（发放、扣除、设置余额）

**用户设置中的 Token 操作**：
- **直接设置余额**：覆盖当前余额，适合修正账面值
- **发放 Token**：增加余额，记录累计获得
- **扣除 Token**：减少余额，记录累计消耗
- 管理员账户不参与 Token 余额限制

### 外链管理

管理外部系统接入：
- 查看接入配置
- 管理认证密钥

### 渠道监控

监控外部渠道运行状态：
- 渠道账号状态
- 消息收发统计
- 异常告警

## 智能体分组

这个分组聚焦智能体能力配置：

### 模型配置

管理可用的 AI 模型：
- 添加新模型（填写 API 密钥、接口地址）
- 配置模型参数
- 设定默认模型

### 线程管理

查看和管理活跃线程：
- 线程状态查看
- 强制取消线程
- 线程详情查看

### 预设智能体

管理预设智能体模板：
- 创建预设智能体（供用户选用）
- 配置预设智能体的工具和模型
- 同步到用户侧

### 工具管理

管理可用工具：
- 查看内置工具列表
- 配置 MCP 外部工具
- 设置工具权限

### 系统提示词

管理全局提示词模板：
- 查看和编辑系统模板
- 切换默认模板包
- 创建新的模板包

## 调试分组

这个分组聚焦工程验证和性能评估：

### 吞吐量测试

- 并发请求测试
- 吞吐量统计

### 性能测试

测试响应性能：
- 单请求延迟测量
- 性能采样

### 蜂群测试

测试蜂群协作能力：
- 蜂群任务测试
- 工蜂协作验证

### 能力评估

评估智能体能力水平：
- 任务完成质量评估
- 能力回归测试

### 调试面板

实时调试工具：
- 实时请求追踪
- 工具调用日志
- 错误排查

### LSP

语言服务协议调试：
- 代码智能服务状态
- LSP 配置验证

## 常用操作流程

### 新用户加入

流程：
1. 用户管理 → 创建用户
2. 设置用户名、密码、所属单位
3. 根据需要发放 Token
4. 用户即可登录使用

### 配置新渠道

流程：
1. 渠道监控 → 添加渠道账号
2. 填写渠道类型和凭证
3. 配置 Webhook 地址（如需要）
4. 查看账号状态确认正常

### 调整系统设置

流程：
1. 系统设置 → 找到对应配置项
2. 修改设置值
3. 保存（通常立即生效）
4. 在内部状态确认效果

### 处理异常会话

流程：
1. 线程管理 → 查看异常线程
2. 查看线程详情和状态
3. 如需终止，点击取消
4. 查看相关日志定位问题

## 快速定位指南

| 我要... | 去哪个面板 |
|---------|------------|
| 添加用户 | 用户管理 |
| 给用户发放 Token | 用户管理 → 点击用户 → 设置 |
| 查看系统运行状态 | 内部状态 |
| 调整系统配置 | 系统设置 |
| 添加新模型 | 模型配置 |
| 管理渠道账号 | 渠道监控 |
| 查看活跃会话 | 线程管理 或 内部状态 |
| 处理卡住的线程 | 线程管理 → 取消 |
| 测试系统性能 | 性能测试 或 吞吐量测试 |
| 查看使用方法 | 使用说明书 |

## 管理端访问方式

管理端通常通过独立地址访问：
- 浏览器打开管理端 URL
- 使用管理员账户登录
- 登录后即可使用各管理面板

具体地址由部署时确定，请询问系统部署人员或查看部署文档。

## 安全提醒

管理端拥有系统最高权限，请注意：

- 只有授权的管理员才能访问
- 管理员账户密码要足够复杂
- 定期更换管理员密码
- 操作前确认影响范围，避免误操作
- 重要操作做好记录
