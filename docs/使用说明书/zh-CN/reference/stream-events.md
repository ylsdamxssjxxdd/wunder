# 会话状态说明

与智能体对话时，你会看到各种状态提示。这页帮你理解每种状态代表什么，以及遇到异常时该怎么处理。

## 状态概览

一个对话从开始到结束，通常会经历以下状态：

```
等待 → 思考/执行 → 完成
          ↓
       需要审批 → 等待你的决定 → 继续
          ↓
       出错/取消 → 结束
```

## 基本状态

### 等待中

**含义**：你的请求已提交，正在排队等待处理。

**何时出现**：
- 系统繁忙时
- 有多个请求同时进行时

**你应该做什么**：等待，通常很快就会开始处理。

### 正在思考

**含义**：智能体正在分析你的请求，准备回答或执行操作。

**这是正常状态**：AI 需要时间来理解你的问题、思考解决方案。

**你应该做什么**：耐心等待，不要重复发送相同消息。

### 正在执行工具

**含义**：智能体正在使用工具完成任务（如读取文件、搜索网页、执行代码等）。

**可能看到的提示**：
- "正在读取文件..."
- "正在搜索..."
- "正在执行..."

**这是正常状态**：复杂任务往往需要多个工具协作完成。

**你应该做什么**：等待执行完成，可以在界面上看到执行进度。

### 已完成

**含义**：智能体已完成你的请求，给出了最终回答。

**这是成功状态**：任务顺利完成。

**你可以做什么**：
- 继续提问
- 查看生成的内容或文件
- 开始新的任务

## 审批状态

### 需要审批

**含义**：智能体想要执行一个需要你确认的操作（如修改重要文件、执行可能有风险的命令等）。

**何时出现**：
- 智能体想修改重要文件
- 智能体想执行敏感命令
- 配置了审批模式时

**你会看到什么**：
- 一个审批请求弹窗或提示
- 智能体想要执行的具体操作说明

**你应该做什么**：
- 仔细阅读操作说明
- 决定是否批准
- 点击"批准"继续，或点击"拒绝"阻止

### 审批已处理

**含义**：你已对审批请求做出了决定。

**两种结果**：
- **你批准了**：智能体继续执行该操作
- **你拒绝了**：智能体放弃该操作，可能寻找其他方案

## 异常状态

### 执行失败

**含义**：智能体在执行过程中遇到了问题，无法完成任务。

**可能的原因**：
- 文件不存在或无法访问
- 网络请求失败
- 工具执行出错
- AI 模型服务异常

**你应该做什么**：
- 查看错误提示了解原因
- 根据提示调整请求（如提供正确的文件路径）
- 重新尝试

### 已取消

**含义**：对话在你或系统主动取消后终止。

**何时出现**：
- 你点击了"停止"按钮
- 系统超时自动取消
- 其他原因导致的主动终止

**你可以做什么**：
- 重新开始对话
- 查看取消前的执行记录

### 会话卡住

**症状**：长时间停留在某个状态，没有任何进展。

**可能的原因**：
- 网络连接中断
- 后端服务异常
- 任务处理超时

**你应该做什么**：
1. 先等待几分钟（有时只是处理时间长）
2. 如果确实卡住，尝试点击"停止"
3. 使用"重置工作状态"功能清理异常状态（在设置里）
4. 如果问题持续，联系管理员或查看故障排查指南

## 连接状态

### 已连接

**含义**：与后端服务的网络连接正常。

**这是正常状态**。

### 连接断开

**含义**：与后端服务的网络连接中断。

**症状**：
- 无法发送新消息
- 状态显示异常
- 提示"连接已断开"

**你应该做什么**：
- 检查网络连接
- 尝试刷新页面或重启应用
- 等待自动重连

## 状态指示器位置

在聊天界面上，你可以在以下位置看到状态提示：

- **对话框上方**：显示当前会话的整体状态
- **消息气泡旁**：显示该消息的处理状态
- **进度区域**：显示工具执行的详细进度
- **蜂群页面**：显示各工蜂的工作进度（如果使用了蜂群协作）

## 常见问题解答

### 为什么有时候响应很慢？

可能原因：
- 任务复杂，需要多步骤执行
- AI 模型服务响应延迟
- 系统负载较高

建议：耐心等待，不要重复发送。

### 为什么智能体要等我审批？

因为该操作可能有风险或影响重要文件。这是安全保护机制，确保敏感操作经过你的确认。

如果你想减少审批，可以与管理员沟通调整审批模式设置。

### 状态显示异常怎么办？

1. 先等待，看是否自动恢复
2. 使用"重置工作状态"功能
3. 刷新页面或重启应用
4. 查看故障排查指南

### 如何查看智能体刚才做了什么？

在对话界面下方或右侧，通常会有执行轨迹或工具调用记录，显示智能体调用了哪些工具、读取了哪些文件、执行了什么操作。
