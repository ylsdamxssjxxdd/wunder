# 形象系统

形象系统让用户可以为智能体设置一个可爱的精灵动画形象。形象会漂浮在屏幕上，根据智能体的状态播放不同的动画，还能显示消息气泡。

## 适用场景

形象适合以下场景：

- **长时间任务**：智能体在执行复杂任务时，形象会显示"运行中"动画，让用户知道它正在工作
- **桌面陪伴**：在桌面端，形象始终可见，像一个小助手陪伴用户的工作
- **多智能体区分**：不同智能体可以设置不同形象，一眼就能区分

## 启用形象

### 桌面端

1. 打开设置面板
2. 找到"形象"设置项
3. 开启"启用形象"开关
4. 从列表中选择一个形象

### 网页端

1. 点击右上角用户头像
2. 进入"偏好设置"
3. 找到"形象"部分
4. 启用并选择形象

## 形象来源

### 全局形象

由系统管理员上传，所有用户可见。这些形象通常经过审核，质量有保证。

### 私有形象

用户自己导入的形象，仅在用户的浏览器中可用。

**导入方法：**

1. 准备一个形象包（ZIP 文件）
2. 在形象设置中点击"导入形象"
3. 选择 ZIP 文件
4. 导入成功后自动选中

## 形象包格式

如果用户想创建自己的形象，需要准备以下内容：

### 文件结构

```
my-companion.zip
├── pet.json        # 清单文件
└── spritesheet.webp # 精灵表图片
```

### 清单文件 (pet.json)

```json
{
  "id": "my-companion",
  "displayName": "我的形象",
  "description": "一个自定义的精灵形象",
  "spritesheetPath": "spritesheet.webp"
}
```

### 精灵表要求

- **格式**：WebP、PNG、GIF 或 JPEG
- **大小**：单个文件最大 18MB
- **包大小**：整个 ZIP 最大 24MB

### 精灵表布局

精灵表应该是一个垂直排列的帧序列，系统会根据状态自动选择帧：

- 第 1 行：idle（静止）
- 第 2 行：running（运行中）
- 第 3 行：waving（挥手）
- 第 4 行：jumping（跳跃）
- 第 5 行：failed（失败）
- 第 6 行：waiting（等待）
- 第 7 行：review（展示）

## 形象交互

### 点击

点击形象会播放挥手动画，并显示形象名称。

### 拖拽

拖拽形象可以改变它在屏幕上的位置。位置会自动保存。

### 右键菜单

右键点击形象打开菜单：

- **打开对话**：跳转到该智能体的聊天
- **隐藏/显示**：切换形象可见性
- **缩放**：调整形象大小（0.5x ~ 1.6x）

## 消息气泡

当智能体回复消息时，形象上方会显示消息气泡：

- 气泡内容是智能体最新回复的摘要
- 气泡会自动消失
- 可以在设置中关闭气泡功能

## 为智能体设置形象

### 方法一：智能体设置面板

1. 打开智能体详情
2. 找到"图标/形象"设置
3. 选择"使用形象"
4. 从下拉列表选择形象
5. 调整显示选项（显示/隐藏、缩放）

### 方法二：编辑智能体配置

在智能体的配置中设置 `icon` 字段：

```json
{
  "icon": {
    "kind": "companion",
    "scope": "global",
    "id": "cat-assistant",
    "show": true,
    "scale": 1.0,
    "messageHints": true
  }
}
```

## 常见问题

### 形象不显示

检查以下几点：

1. 形象功能是否已启用
2. 是否选择了形象
3. 该智能体的形象是否被设置为隐藏
4. 浏览器是否支持 WebGL（某些旧浏览器可能有问题）

### 导入失败

常见原因：

- ZIP 文件超过 24MB
- 精灵表图片超过 18MB
- `pet.json` 格式错误
- 缺少必需字段（id、displayName、spritesheetPath）

### 形象位置重置

形象位置保存在浏览器本地存储中。如果清除浏览器数据，位置会重置到默认位置。

## 管理员操作

如果用户是系统管理员，可以通过管理后台上传全局形象：

1. 进入"系统设置" → "形象管理"
2. 点击"上传形象"
3. 选择形象包 ZIP 文件
4. 上传后所有用户可见

也可以编辑形象的显示名称和描述，或删除不需要的形象。
