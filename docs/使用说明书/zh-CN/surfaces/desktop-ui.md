# 桌面端界面

桌面端是个人用户的主要入口。它不只是网页的"壳"，而是本地优先的完整工作台。

## 桌面端特点

### 本地优先

- 数据存本地，隐私性强
- 不依赖服务器，开箱即用

### 本机文件访问

可以直接访问本机文件系统（在安全边界内），比纯网页端更灵活。

### 自带运行时

安装包自带 Python 运行时，减少首次配置成本。

### 一键重置

异常时可以一键重置工作状态，快速恢复。

## 界面结构

桌面端复用用户端的三栏布局，额外增加桌面特有的功能：

- **左栏**：导航（聊天、文件、智能体、工具、设置、桌面设置、重置）
- **中栏**：列表区
- **右栏**：工作区

## 桌面特有功能

### 桌面运行时设置

配置本地运行环境：Python 路径、本地工作目录、本地工具配置。

### 一键重置工作状态

**什么时候用**：
- 会话卡在运行中
- 蜂群任务状态不一致
- 界面状态和实际不符

**重置后会**：
- 停止所有运行中的会话
- 清空排队任务
- 终止蜂群任务
- 重建干净的线程

**保留的内容**：
- 技能、知识库等长期资产
- 用户配置
- 工作区文件
- 会话历史记录

所以重置是"清理工作状态"，不是"恢复出厂设置"。

### 本地文件能力

桌面端可以让智能体：
- 读取本机文件
- 写入本机文件
- 执行本地命令

这是比网页端更强的能力，但受安全边界限制。

## 什么时候用桌面端

**适合**：
- 个人日常使用
- 需要处理本机文件
- 不想依赖服务器
- 注重隐私和数据本地化

**如果需要**：
- 多用户协作
- 管理员后台
- 组织级部署

请直接用浏览器访问 Server 的网页端。

## 常见问题

### 数据在哪里？

- 工作区：本地工作目录
- 会话历史：本地 SQLite 数据库
- 配置：本地配置文件

### 本地模式能联网吗？

可以。本地模式只是数据存本地，不影响联网访问 AI 模型或网页。

### 能迁移到其他电脑吗？

可以。复制工作目录和数据库文件，在新电脑上配置路径即可。
