# 管理端界面

管理端是系统管理员的后台工作台。普通用户不会看到这个界面。

## 什么时候需要管理端

- 需要配置模型和 API Key
- 需要管理用户、单位、权限
- 需要接入外部渠道（飞书、微信等）
- 需要管理工具和技能
- 需要查看系统运行状态

## 核心功能

### 模型配置

- 添加、编辑、删除模型
- 配置 API Key 和端点
- 测试模型连接
- 设置默认模型

### 预设智能体

- 管理预设智能体卡片
- 配置头像、名称、描述
- 设置模型、工具、提示词
- 管理内置技能

### 工具管理

- 查看所有可用工具
- 启用 / 禁用工具
- 管理技能文件
- 配置 MCP / A2A 服务

### 用户与单位

- 用户列表与状态管理
- 单位（组织）管理
- 权限与角色分配
- 令牌管理

### 渠道管理

- 配置飞书、企业微信、QQ 等渠道
- 管理渠道接入凭证
- 查看渠道运行状态

### 系统监控

- 服务健康状态
- 性能指标
- 日志查看

## 访问方式

管理端通常在 Server 部署后通过浏览器访问，默认端口 18000。
