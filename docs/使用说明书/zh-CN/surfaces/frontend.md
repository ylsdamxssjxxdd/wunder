# 用户端界面

用户端是你的主要工作台。所有日常工作——对话、管理文件、配置智能体、使用工具——都在这里完成。

## 三栏布局

```
┌────────────┬──────────────────┬────────────────────────────┐
│   左栏     │      中栏        │          右栏              │
│   导航     │      列表        │         工作区             │
├────────────┼──────────────────┼────────────────────────────┤
│ • 聊天     │ 会话列表         │ 对话内容                   │
│ • 文件     │ 文件列表         │ 文件预览/详情              │
│ • 智能体   │ 智能体列表       │ 智能体设置                 │
│ • 工具     │ 工具列表         │ 工具详情                   │
│ • 定时任务 │ 任务列表         │ 任务详情                   │
│ • 设置     │ 设置分类         │ 设置内容                   │
│ • 帮助     │ 文档目录         │ 文档内容                   │
└────────────┴──────────────────┴────────────────────────────┘
```

右栏承载主要工作：对话、文件预览、设置面板、帮助文档。

## 聊天功能

### 输入区能力

不只是文本输入框，是一个"上下文装配器"：

| 能力 | 说明 |
|------|------|
| **文本输入** | 支持普通提问和斜杠命令 |
| **文件上传** | 点击上传按钮或拖拽文件 |
| **图片上传** | 直接作为多模态上下文 |
| **文档上传** | 自动转换为 Markdown |
| **音频上传** | 自动转写为文字 |
| **视频上传** | 按帧率提取画面分析 |

### 斜杠命令

| 命令 | 作用 |
|------|------|
| `/new` | 创建新线程 |
| `/stop` | 停止当前执行 |
| `/compact` | 主动压缩对话 |
| `/help` | 打开帮助文档 |

### 保护机制

- **附件处理中**：发送按钮禁用，等待处理完成
- **线程运行中**：新建线程按钮禁用，等待完成或先停止

## 文件功能

### 工作区浏览

- 查看目录结构
- 上传新文件
- 下载文件
- 删除不需要的文件

### 文件预览

- 文本文件：直接预览内容
- 图片：显示图片
- PDF/文档：转换后预览
- 其他：显示基本信息

## 智能体管理

### 查看智能体

- 你的所有智能体列表
- 每个智能体的模型和工具配置
- 当前使用的提示词模板

### 配置智能体

- 选择 AI 模型
- 配置可用工具
- 选择提示词模板
- 添加专属记忆

### 新建线程

- 每个智能体可以有多个线程
- 切换线程继续之前的对话
- 创建新线程开始新话题

## 工具与技能

### 工具面板

- 查看可用工具列表
- 了解每个工具的功能
- 配置技能（能力包）
- 添加知识库

### 知识库

- 让智能体参考你的专属资料
- 支持多种文档格式
- 可检索的知识来源

## 定时任务

- 创建定时执行的任务
- 查看任务执行历史
- 管理、暂停、删除任务
- 手动触发一次执行

## 我的概况与设置

### 我的概况

显示你的个人资料：
- 头像、用户名、等级
- 经验进度条
- 使用统计（会话数、工具调用数、Token 消耗）
- Token 账户余额和趋势

### 账户管理

- 修改用户名
- 修改邮箱
- 修改密码
- 查看所属单位

### 系统设置

- 界面语言切换
- 重置工作状态（异常恢复）

### 帮助手册

- 内嵌文档入口
- 不离开工作台即可查阅

## 界面使用技巧

### 快速导航

左栏是主要入口。点击切换不同功能区。

### 会话管理

中栏列表可以：
- 快速切换会话
- 搜索历史会话
- 创建新会话

### 右栏工作

右栏是主要工作区：
- 对话在这里进行
- 文件在这里预览
- 设置在这里修改

### 状态识别

注意状态指示器：
- 运行中图标 = 正在执行
- 等待图标 = 需要你输入或审批
- 完成图标 = 可以继续
