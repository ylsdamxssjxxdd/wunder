# 用户端界面

用户端是你的主要工作台。对话、管理文件、配置智能体、使用工具——都在这里。

## 三栏布局

- **左栏**：导航入口（聊天、文件、智能体、工具、定时任务、设置、帮助）
- **中栏**：列表区（会话列表、文件列表、智能体列表等）
- **右栏**：工作区（对话、文件预览、设置面板等）

## 聊天

### 输入区

不只是文本框，你可以：

| 能力 | 说明 |
|------|------|
| **文本输入** | 普通提问或斜杠命令 |
| **文件上传** | 点击上传或拖拽文件 |
| **图片上传** | 直接作为多模态输入 |
| **文档上传** | 自动转换为 Markdown |
| **音频上传** | 自动转写为文字 |
| **视频上传** | 按帧率提取画面分析 |

### 斜杠命令

| 命令 | 作用 |
|------|------|
| `/new` | 创建新线程 |
| `/stop` | 停止当前执行 |
| `/compact` | 主动压缩对话 |
| `/help` | 打开帮助文档 |

### 保护机制

- **附件处理中**：发送按钮禁用，等处理完再发
- **线程运行中**：新建线程按钮禁用，等完成或先停止
- **编排态中**：编排线程在普通聊天页只读，需回编排页操作

## 编排工作台

编排页适合需要持续推进的蜂群任务：

- 母蜂持续统筹多个工蜂
- 按轮次查看消息、态势与产物
- 从旧轮次继续并生成新分支

和普通蜂群页相比，编排页更强调"持续推进"，而不是一次性协作。

详见 [编排功能](/docs/zh-CN/surfaces/orchestration/)。

## 文件

- 浏览目录结构
- 上传、下载、删除文件
- 预览文本、图片、PDF 等格式

## 智能体

- 查看你的所有智能体
- 配置模型、工具、提示词
- 添加专属记忆
- 每个智能体可以有多个线程

## 工具与技能

- 查看可用工具列表
- 配置技能（能力包）
- 添加知识库，让智能体参考你的专属资料

## 定时任务

- 创建定时执行的任务
- 查看执行历史
- 管理、暂停、删除任务
- 手动触发一次执行

## 我的概况与设置

### 我的概况

- 头像、用户名、等级
- 经验进度条
- 使用统计（会话数、工具调用数、Token 消耗）
- Token 账户余额和趋势

### 账户管理

- 修改用户名、邮箱、密码
- 查看所属单位

### 系统设置

- 界面语言切换
- 一键重置工作状态（异常恢复）

### 帮助手册

- 内嵌文档入口，不离开工作台即可查阅

## 状态识别

注意界面上的状态指示：

- 🔄 运行中 = 正在执行
- ⏳ 等待中 = 需要你输入或审批
- ✅ 已完成 = 可以继续
