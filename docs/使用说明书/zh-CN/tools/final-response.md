# 最终回复

输出最终答案，结束当前执行轮次。

---

## 工具说明

`最终回复` 是一个特殊的工具，用于：
- 标记当前轮次执行完成
- 向用户输出最终答案
- 结束工具调用链

---

## 参数说明

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `content` | string | ✅ | 最终答案内容 |

---

## 使用示例

```json
{
  "content": "根据分析，我建议采用以下方案：\n1. 首先重构核心模块\n2. 然后添加测试用例\n3. 最后进行性能优化"
}
```

---

## 注意事项

- 这是智能体完成任务时的最后一步
- 调用此工具后，当前轮次结束
- 结果会流式推送给用户
