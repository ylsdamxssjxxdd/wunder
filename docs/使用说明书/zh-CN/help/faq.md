# FAQ

## 必须先注册账号才能用吗？

不需要。你可以用任意名称直接开始对话，系统会自动为你创建身份。

## 页面上显示的 Token 数就是我的费用吗？

不是。Token 数只是当前对话的上下文长度参考，实际费用取决于你使用的模型服务商的计费规则。

## 每次发消息，智能体都会重新学习一遍我的要求吗？

不会。你在对话开头设定的要求会被系统记住并锁定，后续消息都会遵守，不需要重复说明。

## 智能体每次对话都会重新读取我的历史记忆吗？

不会。你的长期记忆只在对话开始时加载一次，不会在每轮对话中反复读取。

## 为什么新建线程按钮有时是灰的？

因为当前智能体还在运行。等它完成，或先停止当前会话再新建。

## 蜂群和子智能体怎么选？

- **蜂群**：让已有的多个智能体协作完成一个任务
- **子智能体**：从当前会话临时派生一个独立运行

## Desktop 必须先部署 Server 吗？

不必须。Desktop 可以本地独立运行。需要多人协作和统一管理时再部署 Server。

## 怎么改用户名或密码？

去「我的概况 → 编辑资料」。改密码需要输入当前密码。

## 忘记密码怎么办？

在登录页点「重置密码」，只需要用户名、邮箱和新密码，不需要旧密码。

## 上传的文件会被自动删除吗？

临时目录里的文件可能会被清理，重要文件请放在工作区持久目录中。

## 为什么有些工具在某个会话里看不到？

可能原因：工具未启用、MCP/A2A 服务未开启、当前智能体没有挂载该工具。

## `web_fetch` 和浏览器工具有什么区别？

- **web_fetch**：抓取网页正文内容
- **浏览器工具**：像真人一样操作网页（点击、滚动、填表）

## 出现问题时先看哪一页？

先看 [故障排查](/docs/zh-CN/help/troubleshooting/)。
