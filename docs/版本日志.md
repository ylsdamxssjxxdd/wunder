# 版本日志

## 版本约定

- 版本号采用语义化版本：`MAJOR.MINOR.PATCH`。
- `MAJOR`：不兼容变更；`MINOR`：向后兼容的新功能；`PATCH`：向后兼容的问题修复。
- 在 `0.x.y` 阶段仍属原型期，功能与接口可能发生调整，不承诺跨小版本完全兼容。
- 预发布版本使用 `-alpha.N`、`-beta.N`、`-rc.N` 后缀；正式版不带后缀。
- 发布记录按时间倒序维护（最新版本在上），每个版本至少包含：发布日期、版本说明、核心能力变化。

## 0.1.0（2026-03-21）

wunder 0.1.0 是首个可发布原型版，形成 server、cli、desktop 三种运行形态。server 提供多租户、用户与单位管理、智能体应用构建发布与统一 `/wunder` 调度入口；系统内置工具链、知识库与长期记忆，支持以 WebSocket 为先、SSE 兜底的流式交互。desktop 面向个人用户，cli 支持本地自动化与工程协同。
