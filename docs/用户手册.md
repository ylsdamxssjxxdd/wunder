# Wille 用户手册（用户侧）

适用范围：Wille 用户侧前端（`/app`）。演示模式可访问 `/demo`，功能与正式模式一致但数据为演示环境。

## 页面入口与导航
- 顶部导航：世界 / 工具管理 / 聊天。
- 右上角：主题切换、进入“我的概况”、退出登录。
- 窄屏：聊天页的“沙盒容器/历史记录”以弹窗方式打开。

## 账号与登录
1. 打开登录页，输入用户名与密码登录。
2. 没有账号点击“注册账号”，按提示填写后即可进入系统。
3. 登录后默认进入“聊天”。

## 世界（智能体应用入口）
- 作用：统一入口管理你的智能体应用，并快速进入对应聊天。
- 列表信息：每个卡片显示名称、描述、工具数量、更新时间；卡片状态会显示“运行中/待选择”。
- 新建智能体应用：
  1) 点击“新建智能体应用”。
  2) 填写名称与描述。
  3) 选择头像图标/颜色（可自定义色值）。
  4) 选择要挂载的工具与技能（支持分组全选/取消）。
  5) 可选：填写“智能体提示词（追加）”。
  6) 保存后生成卡片。
- 编辑/删除：卡片右上角按钮进入编辑或删除。
- 共享：编辑中打开“共享”，同等级用户可见（是否展示共享区域由系统配置决定）。
- 搜索：搜索框支持按名称/描述/工具关键字过滤。
- 进入聊天：点击卡片进入该智能体的聊天；“通用聊天”为默认入口。

## 聊天
- 新建会话：点击顶部“新建会话”。
- 历史记录：左侧列表切换会话；点击垃圾桶删除会话。
- 发送消息：输入后发送；支持拖拽/上传附件。
- 附件支持：常见图片与文档/代码文件，系统会先解析再允许发送。
  - 图片：png/jpg/jpeg/gif/bmp/webp/svg
  - 文档/代码：txt/md/markdown/html/htm/py/c/cpp/cc/h/hpp/json/js/ts/css/ini/cfg/log/doc/docx/odt/pptx/odp/xlsx/ods/wps/et/dps
- 中止回复：回复过程中点击停止按钮。
- 问询面板：出现“待选择”时先勾选选项，再发送。
- 会话工具调整：
  - 点击顶部“调整工具”图标，勾选/取消本会话工具。
  - 点击“恢复默认”回到智能体默认工具配置。
  - 点击“应用”生效；仅影响当前会话。
- 系统提示词与能力：欢迎消息右侧眼睛按钮可预览系统提示词；悬浮查看工具/技能能力列表。
- 刷新/断线：页面刷新会自动加载最近会话，并尝试续传未完成回复；若续传失败可继续追问。

## 沙盒容器（工作区）
- 作用：为当前智能体提供隔离的文件空间，工具产物与上传文件会落在这里。
- 说明：沙盒容器与工作区为同一空间，看到的文件完全一致。
- 入口：聊天页左侧“沙盒容器”，或窄屏顶部按钮（如有“工作区”页面入口，内容与此处一致）。
- 主要操作：
  - 上传/拖拽上传、刷新、清空、打包全量下载。
  - 目录树展开/收起；支持搜索文件名。
  - 右键菜单：新建文件/文件夹、重命名、移动、复制、删除、下载。
  - 双击文件预览；文本文件可编辑并保存。
- 预览规则：
  - 图片与 PDF 可内嵌预览；Office 文件提示下载后本地打开。
  - 超大文本会提示“内容已截断”，建议下载查看。

## 工具管理
### 管理员开放工具（只读）
- 查看系统提供的内置工具、MCP/A2A 工具、技能与知识库工具。

### MCP 工具
1. 新增服务：填写名称、地址、传输类型与可选 Headers。
2. 连接/刷新：点击“连接”加载工具列表；已连接服务可“全部刷新”。
3. 启用/共享：
   - 勾选“启用”后工具可被会话调用。
   - 勾选“共享”后他人可在“共享工具”中看到。

### 技能
- 上传技能包（zip）后刷新列表。
- 勾选“启用”后可用；勾选“共享”后他人可见。
- 点击技能可查看 SKILL.md 详情。

### 知识库
- 新增知识库时选择类型：字面 / 向量。
- 字面知识库：
  - 上传或新建 `.md` 文件；支持在线编辑并保存。
- 向量知识库：
  - 上传文档后生成切片；可查看原文、批量嵌入、批量删除。
  - 支持“重建索引”与“知识库测试”。
  - 向量库不支持直接编辑原文文件。

### 共享工具
- 这里的勾选会把他人共享的工具加入你的工具池。
- 勾选后才会出现在智能体工具列表中。

## 我的概况
- 查看账号信息、对话统计、额度与上下文占用。
- 点击“编辑”可更新用户名/邮箱。
