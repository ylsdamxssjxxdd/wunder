# Wille 用户手册（用户侧）

适用范围：Wille 用户侧前端（`/app`）。演示模式可访问 `/demo`。

## 快速开始
1. 打开登录页，输入用户名/密码登录；没有账号点击“注册账号”。
2. 登录后默认进入“聊天”，顶部导航可切换“广场 / 工具管理 / 聊天”。
3. 右上角可切换浅色/深色主题，点击用户信息进入“我的概况”。

## 功能广场（智能体应用）
- 新建：点击“新建智能体应用”，填写名称/描述，选择工具与技能后保存。
- 进入：点击卡片进入对应聊天；“通用聊天”为默认入口。
- 编辑/删除：卡片右上角的编辑或删除按钮。
- 共享：编辑中开启“共享”后，同等级用户可见（是否展示共享区域由系统配置决定）。
- 搜索：使用搜索框按名称/描述/工具关键字过滤。

## 聊天
- 新建会话：点击顶部“新建会话”。
- 发送消息：输入后发送；支持拖拽/上传图片与文档附件（解析完成后再发送）。
- 中止回复：回复过程中点击停止按钮。
- 问询面板：出现“待选择”时，先勾选选项再发送。
- 调整工具：点击顶部“调整工具”图标，勾选本会话工具或“恢复默认”。
- 系统提示词/能力：欢迎消息右侧眼睛图标可预览系统提示词；悬浮查看工具/技能列表。
- 历史记录：左侧选择会话；点击垃圾桶删除。窄屏使用“历史记录”按钮。
- 资源卡片：回复中的文件支持下载，图片可预览。
- 刷新页面：自动加载最近会话，并尝试续传未完成回复。

## 临时文件区（聊天内）
- 入口：聊天页左侧栏或顶部“临时文件”。
- 常用操作：上传/下载、清空、搜索，右键新建/重命名/移动/复制/删除，双击预览或编辑文本文件。

## 工具管理
- 管理员开放工具（只读）：查看系统可用工具。
- MCP 工具：新增/导入服务 → 连接 → 勾选启用/共享。
- 技能：上传技能包（zip）→ 勾选启用/共享 → 点击查看详情。
- 知识库：新增知识库（字面/向量）。
  - 字面：上传或新建 `.md`，编辑后保存。
  - 向量：上传文档后查看切片，可重建索引、批量嵌入/删除、测试检索。
- 共享工具：在这里勾选的共享工具，才会出现在智能体工具列表里。

## 工作区（用户工作区）
- 文件夹树 + 文件列表 + 右侧预览信息。
- 操作：新建文件夹、上传、下载、删除；点击列表行查看详情。

## 我的概况
- 查看账号信息、对话统计、额度与上下文占用。
- 点击“编辑”可更新用户名/邮箱。
