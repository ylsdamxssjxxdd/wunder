自建工具：个人专属能力

每个人都能拥有自己的工具箱

是什么
  • 个人工具包独立配置
  • 路径：data/user_tools/<user_id>
  • 别名统一为 user_id@tool
有什么用
  • 满足个性化需求且不影响他人
  • 能力可选择共享给团队
治理要点

隔离可控，启用需明确勾选

共享工具：团队复用能力

能力共享，但工作区仍独立

是什么
  • 共享工具清单与配置
  • 不共享对方工作区
  • 命名统一为 owner_id@tool
有什么用
  • 成熟能力快速复制到团队
  • 降低协作门槛与沟通成本
治理要点

使用者需显式启用共享工具

第4节 上下文压缩
上下文压缩

上下文压缩：让长对话持续可用

对话越长越重，需要稳定机制

触发条件
  • 上下文占用 token 达阈值
  • 当前消息接近安全上限
处理方式
  • 保留系统提示词 + 最近消息
  • 生成结构化摘要并注入
  • 重置历史占用继续对话
口径说明

统计的是上下文占用 token，不是总消耗
