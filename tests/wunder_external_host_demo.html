Wunder 外链宿主页

这个页面模拟“外部系统里嵌一个 Wunder 入口”的最小宿主。填入 `user_id` 和可选 `agent_name` 后，会生成 `http://127.0.0.1:18001/login?...`，你可以直接 iframe 预览、新窗口打开，或者复制给其他本地页面接入。

测试控制栏

生成链接

当前按本地 `18001` 单端口入口生成。若你改成本地前端直出 `18002`，这里也会同步切换。

  • 本链路当前设计为本地/桌面免 token 外链，适合你现在接入 wunder 看效果，不适合作为公网匿名入口直接暴露。
  • 如果 `agent_name` 未命中已有智能体，会自动回退到默认聊天入口。
  • 如果新窗口能打开而 iframe 白屏，优先检查浏览器控制台是否有嵌入安全头报错。
